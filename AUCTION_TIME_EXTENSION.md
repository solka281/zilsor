# Система продления времени аукционов

## 🕐 Новая механика:
Если до конца аукциона осталось меньше 5 минут и кто-то делает ставку, аукцион автоматически продлевается на 5 минут от текущего времени.

## ⚡ Как работает:

### Условие продления:
- До конца аукциона осталось **меньше 5 минут**
- Игрок делает новую ставку
- Ставка успешно принята

### Результат:
- Время окончания аукциона устанавливается на **+5 минут от текущего времени**
- Игрок получает уведомление о продлении
- Все участники видят обновленное время

## 🔧 Техническая реализация:

### Бэкенд (marketplace_api.js):
```javascript
// Проверяем, нужно ли продлить аукцион
const now = new Date();
const endsAt = new Date(auction.ends_at);
const timeLeft = endsAt - now;
const fiveMinutes = 5 * 60 * 1000; // 5 минут в миллисекундах

let newEndsAt = auction.ends_at;
let timeExtended = false;

if (timeLeft < fiveMinutes) {
  // Продлеваем аукцион на 5 минут от текущего времени
  newEndsAt = new Date(now.getTime() + fiveMinutes).toISOString();
  timeExtended = true;
}

// Обновляем время окончания в базе данных
UPDATE auctions SET ends_at = ? WHERE id = ?
```

### Фронтенд (marketplace/app.js):
```javascript
if (result.timeExtended) {
  message += ' ⏰ Аукцион продлен на 5 минут!';
}
```

## ⚙️ Настройки:

### Константы:
- **Время продления:** 5 минут (300 секунд)
- **Порог для продления:** 5 минут до конца
- **Максимальные продления:** Без ограничений

### Возможные улучшения:
- Ограничение количества продлений
- Уменьшение времени продления с каждым разом
- Настройка порога продления для разных типов аукционов
