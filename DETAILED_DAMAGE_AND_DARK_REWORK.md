# 📊 Детализация урона + новый эффект тьмы

## 🌑 НОВЫЙ ЭФФЕКТ АМУЛЕТА ТЬМЫ

### Что изменилось:
**Было:** Проклятие (-15% к характеристикам на 3 хода)
**Стало:** Поглощение жизни (+25% урона + лечение атакующего)

### 🌑 Поглощение жизни:
- **Шанс**: 15%
- **Дополнительный урон**: +25% от базового урона
- **Лечение**: 50% от дополнительного урона
- **Пример**: Урон 100 → +25 урона → +12 HP атакующему

### 💡 Почему лучше:
- **Активный эффект** вместо пассивного ослабления
- **Мгновенная польза** для атакующего
- **Тематически подходит** тьме (вампиризм)
- **Не фрустрирует** противника долгими дебаффами

---

## 📊 ДЕТАЛИЗИРОВАННЫЙ УРОН

### Новый формат сообщений:
```
solkauwu: 295 урона (Атака 85 + Дракон 68 + Пламенный клинок 25 + Руна берсерка 50 + Амулет огня 67)
```

### 📋 Виды урона с названиями:
- **Атака X** - базовый урон от характеристик
- **Крит X** - дополнительный урон от критического удара
- **[Название расы] X** - урон от расовой способности
- **[Название предмета] X** - урон от конкретного предмета
- **[Название амулета] X** - урон от элементального эффекта

### 🎯 Примеры детализации:
```
Игрок: 150 урона (Атака 100 + Крит 50)
Игрок: 200 урона (Атака 85 + Дракон 40 + Экскалибур 75)
Игрок: 180 урона (Спец.атака 120 + Руна берсерка 60)
Игрок: 220 урона (Атака 100 + Амулет тьмы 25 + Пламенный клинок 30 + Крит 65)
```

## 🔧 ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ

### Обновленная структура damageBreakdown:
```javascript
damageBreakdown: {
  base: 85,                              // базовый урон
  critical: 0,                           // урон от крита
  ability: { damage: 68, name: 'Дракон' }, // способность расы
  elemental: { damage: 25, name: 'Амулет тьмы' }, // элементы
  items: [                               // массив предметов
    { name: 'Пламенный клинок', damageBonus: 25 },
    { name: 'Руна берсерка', damageBonus: 50 }
  ],
  total: 295                             // итоговый урон
}
```

### Новые функции:
- `getItemEffectDetails()` - получает названия предметов по эффектам
- `getTriggeredElementName()` - определяет какой амулет сработал

### Карта предметов:
```javascript
const itemMap = {
  'fire_damage_25': { name: 'Пламенный клинок', damageBonus: 25 },
  'shadow_strike': { name: 'Клинок теней', damageBonus: 0 },
  'holy_damage_50': { name: 'Экскалибур', damageBonus: 50 },
  'berserk_mode': { name: 'Руна берсерка', damageBonus: 50 }
}
```

## 🎮 ВЛИЯНИЕ НА ГЕЙМПЛЕЙ

### ✅ Преимущества детализации:
- **Полная прозрачность** - игроки видят вклад каждого предмета
- **Лучшее понимание билдов** - можно оценить эффективность экипировки
- **Тактическая глубина** - ясно какие предметы работают в бою
- **Мотивация к улучшению** - видно что дает каждый апгрейд

### 🌑 Преимущества нового эффекта тьмы:
- **Активная польза** вместо пассивного ослабления
- **Синергия с уроном** - чем больше урон, тем больше лечение
- **Тематическая уместность** - тьма поглощает жизнь
- **Баланс** - не слишком сильный, но полезный эффект
