# Как запустить Wiki

## Шаг 1: Запустите сервер

Откройте терминал в папке `wiki` и выполните:

```bash
cd wiki
python -m http.server 8000
```

Или если у вас Python 2:
```bash
python -m SimpleHTTPServer 8000
```

## Шаг 2: Откройте в браузере

Откройте браузер и перейдите по адресу:
```
http://localhost:8000/
```

## Шаг 3: Навигация

### Чтобы увидеть расы:
1. В левом меню найдите раздел "Расы"
2. Кликните на "Все расы (23)"
3. Или кликните на большую кнопку "Расы" на главной странице

### Чтобы увидеть предметы:
1. В левом меню найдите раздел "Предметы"  
2. Кликните на "Все предметы"
3. Или кликните на большую кнопку "Предметы" на главной странице

### Чтобы увидеть локации:
1. В левом меню найдите раздел "Локации"
2. Кликните на "Все локации"

## Альтернативный способ - прямые ссылки:

- Расы: http://localhost:8000/races.html
- Предметы: http://localhost:8000/items.html
- Главная: http://localhost:8000/index.html

## Если расы не показываются:

1. Откройте консоль браузера (F12)
2. Проверьте есть ли ошибки
3. Убедитесь что файл `app.js` загрузился
4. Попробуйте открыть напрямую: http://localhost:8000/races.html

## Структура файлов:

```
wiki/
├── index.html          # Главная страница с навигацией
├── races.html          # Страница со всеми расами (работает отдельно)
├── items.html          # Страница со всеми предметами (работает отдельно)
├── app.js              # JavaScript для навигации на главной странице
├── styles.css          # Стили
├── images/
│   └── races/          # Картинки рас (23 файла)
└── ...
```

## Проверка что всё работает:

1. ✅ Сервер запущен - видите "Serving HTTP on 0.0.0.0 port 8000"
2. ✅ Главная страница открывается - http://localhost:8000/
3. ✅ Расы показываются - кликните "Все расы (23)" в меню
4. ✅ Картинки загружаются - должны быть видны изображения рас

## Если что-то не работает:

### Проблема: Расы не показываются на главной странице
**Решение**: Откройте напрямую http://localhost:8000/races.html

### Проблема: Картинки не загружаются
**Решение**: Убедитесь что папка `wiki/images/races/` содержит файлы:
- human.jpg
- elf.jpg
- dwarf.jpg
- и т.д. (всего 23 файла)

### Проблема: Навигация не работает
**Решение**: 
1. Проверьте что файл `app.js` загрузился (F12 → Network)
2. Обновите страницу (Ctrl+F5)
3. Попробуйте другой браузер

## Быстрый тест:

Откройте в браузере эти ссылки по очереди:
1. http://localhost:8000/races.html - должны показаться все расы
2. http://localhost:8000/items.html - должны показаться все предметы
3. http://localhost:8000/index.html - главная страница

Если эти страницы работают - всё ОК!
