## 🔧 Как запустить

### Вариант 1: Прямое открытие
```bash
# Просто откройте wiki/index.html в браузере
```

### Вариант 2: Локальный сервер
```bash
cd wiki
python -m http.server 8000
# Откройте http://localhost:8000
```

### Вариант 3: Live Server (VS Code)
```
1. Установите расширение "Live Server"
2. Правый клик на wiki/index.html
3. "Open with Live Server"
```

## 🐛 Решение проблем

### "Загрузка данных..." не исчезает
**Решение**: Убедитесь что файлы `race_data.js` и `item_data.js` загружены ДО `app.js` в HTML

### Изображения не отображаются
**Решение**: Скопируйте изображения из `../images/races/` в `wiki/images/races/`

### Навигация не работает
**Решение**: Проверьте консоль браузера (F12) на ошибки JavaScript
