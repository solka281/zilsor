## 🎯 Как работает уклонение:

### Базовая формула:
```javascript
// Базовый шанс уклонения
const baseDodgeChance = 0.05 + (defender.defense / 1000);

// С учетом предметов
const finalDodgeChance = modifyDodgeWithItems(baseDodgeChance, itemEffects);
```

## 📊 Источники уклонения:

### 1. **Базовое уклонение:**
- **5% базовый шанс** для всех персонажей
- **+1% за каждые 1000 защиты** (defense / 1000)

### 2. **Бонусы от предметов:**

#### 🟣 Сапоги теней (EPIC)
- **Эффект:** `dodge_20`
- **Бонус:** +20% к уклонению
- **Итого:** ~25% шанс уклониться

#### 🟣 Руна скорости (EPIC)  
- **Эффект:** `speed_boost_25`
- **Бонус:** +25% к уклонению
- **Итого:** ~30% шанс уклониться

#### 🔵 Сапоги скорости (RARE)
- **Эффект:** `speed_boost_15`
- **Бонус:** +15% к уклонению
- **Итого:** ~20% шанс уклониться

#### 🟣 Титановые поножи (EPIC)
- **Эффект:** `speed_boost_10`
- **Бонус:** +10% к уклонению
- **Итого:** ~15% шанс уклониться

## 🔢 Примеры расчетов:

### Пример 1: Базовый персонаж
- Защита: 100
- Базовое уклонение: 5% + (100/1000) = **6%**

### Пример 2: С Сапогами теней
- Защита: 100
- Базовое: 6%
- Бонус от предмета: +20%
- **Итого: 26% шанс уклониться**

### Пример 3: Танк с высокой защитой
- Защита: 500
- Базовое: 5% + (500/1000) = 10%
- С Руной скорости: +25%
- **Итого: 35% шанс уклониться**

### Пример 4: Максимальное уклонение
- Защита: 1000 (10% базовых)
- Сапоги теней: +20%
- Руна скорости: +25%
- **Итого: 55% шанс уклониться!**

## ⚔️ Как это работает в бою:

### При атаке:
1. Рассчитывается шанс уклонения защищающегося
2. Генерируется случайное число (0-1)
3. Если число меньше шанса уклонения → уклонение
4. Показывается сообщение: "💨 [Имя] уклонился от атаки!"

### Код проверки:
```javascript
if (Math.random() < dodgeChance) {
  return { dodged: true, damage: 0 };
}
```

## 🎮 Стратегии использования:

### 🏃 Скоростной билд (Dodge Tank):
- **Сапоги теней** (+20% уклонение)
- **Руна скорости** (+25% уклонение)  
- **Высокая защита** (бонус от статов)
- **Результат:** 40-55% уклонения

### ⚖️ Сбалансированный билд:
- **Сапоги скорости** (+15% уклонение)
- **Средняя защита**
- **Результат:** 20-25% уклонения

### 🛡️ Танк билд:
- **Высокая защита** (500+)
- **Руна скорости** (+25% уклонение)
- **Результат:** 35%+ уклонения + высокая защита

## 📈 Важные моменты:

### ✅ Преимущества уклонения:
- **Полное избежание урона** (0 урона при уклонении)
- **Не тратится HP** на блокировку/поглощение
- **Работает против любого урона** (физический, магический)

### ❌ Ограничения:
- **Зависит от RNG** (случайности)
- **Не защищает от эффектов** (яды, дебаффы могут пройти)
- **Нет гарантий** (даже 90% не дает 100% защиты)

## 🎯 Вывод:

**НЕТ отдельного поля "уклонение" в базе данных!**

Уклонение рассчитывается динамически на основе:
1. **Защиты персонажа** (из базы данных)
2. **Эффектов экипированных предметов** (из базы данных)
3. **Формулы в коде** (5% + defense/1000 + бонусы предметов)

Это позволяет гибко настраивать баланс без изменения структуры БД! 🚀
