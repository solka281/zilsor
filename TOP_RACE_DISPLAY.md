# Отображение расы в топе игроков

## 🎯 Изменение:

Добавлено отображение расы игрока рядом с его ником в топе.

## 📊 Новый формат:

**Было:**
```
🥇 💎 Игрок
   🌲 Лес: 25 | ⚡ 1200 | 💰 50000
```

**Стало:**
```
🥇 💎 Игрок 🟡Феникс
   🌲 Лес: 25 | ⚡ 1200 | 💰 50000
```

## 🎨 Элементы отображения:

1. **Медаль** - 🥇 🥈 🥉 или номер места
2. **VIP значок** - 💎 (если есть VIP)
3. **Имя игрока** - отображаемое имя или username
4. **Иконка редкости расы** - ⚪️ 🔵 🟣 🔴 🟡 ⚫️
5. **Название расы** - название расы игрока
6. **Статистика** - уровень леса, сила, золото

## 📝 Примеры:

```
🏆 Топ игроков

🥇 💎 ПроИгрок 🟡Феникс
   🌲 Лес: 50 | ⚡ 2500 | 💰 100000

🥈 Игрок123 🔴Демон
   🌲 Лес: 45 | ⚡ 2200 | 💰 85000

🥉 НовичОк ⚪️Человек
   🌲 Лес: 30 | ⚡ 1500 | 💰 50000

4. Воин 🔵Эльф
   🌲 Лес: 28 | ⚡ 1400 | 💰 45000
```

## 🔧 Технические детали:

```javascript
const rarityIcon = player.race_rarity ? config.RARITY[player.race_rarity].color : '⚪️';
const raceName = player.race_name || 'Без расы';

message += `${medal} ${vipIcon}${displayName} ${rarityIcon}${raceName}\n`;
```

## 📍 Где применяется:

- ✅ Кнопка "🏆 Топ игроков" в меню рейтинга
- ✅ Команда `/top` в чате

## 💡 Преимущества:

- Игроки видят какие расы у топовых игроков
- Мотивация получить редкую расу
- Больше информации о конкурентах
- Красивое визуальное оформление

## ✅ Статус:
**ОБНОВЛЕНО!** Раса теперь отображается рядом с ником в топе игроков.
