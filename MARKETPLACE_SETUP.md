# 🏪 Маркетплейс Zilsor Race

## Описание

Telegram Mini App для торговли предметами между игроками в игре Zilsor Race. Включает в себя:

- 🛒 **Маркетплейс** - покупка и продажа предметов
- 🎒 **Инвентарь** - управление своими предметами
- ⏰ **Аукционы** - создание и участие в аукционах
- 💱 **Обмен валют** - конвертация золота в кристаллы и обратно

## Функции

### 🛒 Маркетплейс
- Просмотр всех предметов на продаже
- Фильтрация по типу предмета и редкости
- Покупка предметов за золото или кристаллы
- Информация о продавце

### 🎒 Инвентарь
- Просмотр всех предметов игрока
- Выставление предметов на продажу
- Создание аукционов
- Защита от продажи экипированных предметов

### ⏰ Аукционы
- Создание аукционов с настраиваемой длительностью
- Система ставок в реальном времени
- Автоматическое завершение аукционов
- История ставок

### 💱 Обмен валют
- Покупка кристаллов за золото (курс: 1000💰 = 1💎)
- Продажа кристаллов за золото (курс: 1💎 = 800💰)
- Мгновенные транзакции

### База данных

#### Таблица `marketplace_items`
Хранит предметы на продаже:
- `seller_id` - ID продавца
- `item_id` - ID предмета
- `price` - цена
- `currency` - валюта (gold/crystals)

#### Таблица `auctions`
Хранит активные аукционы:
- `seller_id` - ID создателя аукциона
- `start_price` - начальная цена
- `current_bid` - текущая ставка
- `ends_at` - время окончания

#### Таблица `auction_bids`
Хранит историю ставок:
- `auction_id` - ID аукциона
- `bidder_id` - ID участника
- `bid_amount` - размер ставки

### API Endpoints

#### Пользователь
- `GET /api/marketplace/user/:userId` - данные пользователя
- `GET /api/marketplace/inventory/:userId` - инвентарь

#### Маркетплейс
- `GET /api/marketplace/items` - предметы на продаже
- `POST /api/marketplace/sell` - выставить на продажу
- `POST /api/marketplace/buy` - купить предмет

#### Аукционы
- `GET /api/marketplace/auctions` - активные аукционы
- `POST /api/marketplace/auction/create` - создать аукцион
- `POST /api/marketplace/auction/bid` - сделать ставку

#### Обмен валют
- `POST /api/marketplace/exchange/buy-crystals` - купить кристаллы
- `POST /api/marketplace/exchange/sell-crystals` - продать кристаллы

## Установка и настройка

### 1. Настройка сервера

1. Убедитесь что установлен Express:
```bash
npm install express
```

2. Запустите бота - сервер маркетплейса запустится автоматически на порту 3000

### 2. Настройка домена

В файле `bot.js` замените `your-domain.com` на ваш реальный домен:
```javascript
const webAppUrl = `https://your-domain.com/api/marketplace/app`;
```

### 3. Настройка Telegram Bot

1. Откройте @BotFather в Telegram
2. Выберите вашего бота
3. Используйте команду `/setmenubutton`
4. Установите Web App URL: `https://your-domain.com/api/marketplace/app`

### 4. SSL сертификат

Telegram Web Apps требуют HTTPS. Настройте SSL сертификат для вашего домена.
