# Отладка системы характеристик

## Проблема:
Пользователь прокачал здоровье 1 раз, но получил только +2 HP вместо ожидаемых +12 HP.

## Ожидаемое поведение:
По формуле здоровья: `10 + level * 2`
- Уровень 1: `10 + 1 * 2 = 12 HP`

## Возможные причины:

### 1. 🔧 Неправильная формула в коде
**Проверить**: `characteristics.js` → `calculateBonus()`
```javascript
// Должно быть:
function calculateBonus(baseBonus, levelMultiplier, level) {
  if (level === 0) return 0;
  return Math.round(baseBonus + (level * levelMultiplier));
}
```

### 2. 📊 Неправильные константы
**Проверить**: `CHARACTERISTIC_TYPES.hp`
```javascript
hp: {
  name: 'Здоровье',
  icon: '❤️',
  base_cost: 80,
  cost_multiplier: 1.18,
  base_bonus: 10,        // ← Должно быть 10
  level_multiplier: 2    // ← Должно быть 2
}
```

### 3. 🗄️ Проблема с базой данных
**Проверить**: Данные в таблице `player_characteristics`
```sql
SELECT * FROM player_characteristics WHERE player_id = [USER_ID];
```

### 4. 🔗 Проблема интеграции с duels.js
**Проверить**: Как бонусы применяются в `calculatePlayerPower()`
```javascript
// В duels.js должно быть:
totalHP += charBonuses.hp;
```

## Как диагностировать:

### 1. Проверить логи бота
После прокачки в консоли должно появиться:
```
🔧 [DEBUG] upgradeCharacteristic вызвана: playerId=123456, type=hp
💰 [DEBUG] Золото игрока: 1000
📊 [DEBUG] Текущий уровень hp: 0
💸 [DEBUG] Стоимость улучшения: 80
✅ [DEBUG] Золото списано: -80
✅ [DEBUG] Характеристика обновлена:
   Тип: hp
   Старый уровень: 0
   Новый уровень: 1
   Формула бонуса: 10 + 1 * 2
   Рассчитанный бонус: 12
✅ Игрок 123456 улучшил hp до уровня 1 (+12)
```

### 2. Запустить скрипт отладки
```bash
node debug_characteristics.js
```

Покажет:
- Формулы расчета
- Данные в базе
- Текущие уровни характеристик всех игроков

### 3. Проверить профиль игрока
В игре должно отображаться:
- **До прокачки**: HP = базовое HP расы
- **После прокачки**: HP = базовое HP расы + 12

## Ожидаемые значения:

### Здоровье (HP):
- Уровень 0: +0 HP
- Уровень 1: +12 HP  
- Уровень 2: +14 HP
- Уровень 3: +16 HP

### Урон (Attack):
- Уровень 0: +0 атаки
- Уровень 1: +7 атаки
- Уровень 2: +8 атаки  
- Уровень 3: +9 атаки

### Защита (Defense):
- Уровень 0: +0 защиты
- Уровень 1: +3 защиты (2.5 округлено)
- Уровень 2: +3 защиты
- Уровень 3: +4 защиты (3.5 округлено)

## Если проблема остается:

1. **Проверить логи** - есть ли отладочная информация
2. **Проверить базу** - сохранился ли правильный уровень
3. **Проверить интеграцию** - применяются ли бонусы в duels.js
4. **Проверить UI** - правильно ли отображается в профиле

Отладочная информация поможет точно определить на каком этапе происходит ошибка!
