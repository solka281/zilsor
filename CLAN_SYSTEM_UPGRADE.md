# Обновление системы кланов

## ✅ СТАТУС: ЗАВЕРШЕНО!

## 🎯 Реализованные возможности:

### 👥 Для всех игроков:
1. ✅ **Просмотр списка кланов** - карточки кланов с навигацией (стрелки ◀️ ▶️)
2. ✅ **Вступление в клан** - одним нажатием (если клан открытый) или отправка заявки
3. ✅ **Выход из клана** - покинуть клан в любой момент
4. ✅ **Отображение аватара клана** - если установлен

### 👑 Для лидера клана:
1. ✅ **Управление заявками** - принимать/отклонять заявки на вступление
2. ✅ **Тип вступления** - переключать между "открытым" (🔓) и "по заявкам" (🔒)
3. ✅ **Кик игроков** - исключать участников из клана
4. ✅ **Изменить лимит** - установить максимальное количество участников (5-100)
5. ✅ **Установить аватар** - загрузить изображение для клана

## 📊 Структура базы данных:

### Таблица `clans`:
```sql
- max_members INTEGER DEFAULT 20  -- Максимум участников
- join_type TEXT DEFAULT 'open'   -- Тип: 'open' или 'request'
- avatar_file_id TEXT             -- ID файла аватара в Telegram
```

### Таблица `clan_requests`:
```sql
CREATE TABLE clan_requests (
  id INTEGER PRIMARY KEY,
  clan_id INTEGER,
  player_id INTEGER,
  status TEXT DEFAULT 'pending',  -- pending/accepted/rejected
  created_at DATETIME
)
```

## 🎮 Реализованные функции:

### В clan_management.js:
1. ✅ `getAvailableClans()` - список всех кланов
2. ✅ `sendJoinRequest()` - отправить заявку/вступить
3. ✅ `getClanRequests()` - получить заявки в клан
4. ✅ `acceptRequest()` - принять заявку
5. ✅ `rejectRequest()` - отклонить заявку
6. ✅ `kickPlayer()` - кикнуть игрока
7. ✅ `changeJoinType()` - изменить тип вступления
8. ✅ `changeMaxMembers()` - изменить лимит участников
9. ✅ `setClanAvatar()` - установить аватар

### В bot.js:
1. ✅ `showClanCard()` - отображение карточки клана с навигацией
2. ✅ `getClanMenu()` - обновленное меню клана с настройками для лидера
3. ✅ Обработчики callback:
   - `clan_page_X` - навигация по кланам
   - `clan_join_X` - вступление/заявка
   - `clan_settings` - меню настроек
   - `clan_view_requests` - просмотр заявок
   - `clan_accept_X` / `clan_reject_X` - управление заявками
   - `clan_change_join_type` - изменение типа вступления
   - `clan_change_max_members` - изменение лимита
   - `clan_set_avatar` - установка аватара
   - `clan_kick_menu` / `clan_kick_confirm_X` - кик игроков
4. ✅ Обработчики текстовых сообщений:
   - Ввод нового лимита участников
5. ✅ Обработчик фото для аватара клана

## 🔄 Логика вступления:

### Открытый клан (open):
```
Игрок нажимает "✅ Вступить" → Сразу добавляется в клан
→ Получает уведомление о вступлении
```

### Клан по заявкам (request):
```
Игрок нажимает "📝 Подать заявку" → Создается заявка
→ Лидер видит заявку в меню настроек
→ Лидер принимает/отклоняет
→ Игрок получает уведомление
```

- ◀️ / ▶️ - навигация по кланам
- ✅ Вступить / 📝 Подать заявку (в зависимости от типа клана)
- 🔙 Назад

### Для участника клана:
- 👥 Члены клана
- 🚪 Покинуть клан
- 🔙 Назад

### Для лидера клана:
- 👥 Члены клана

  - 📋 Заявки (X) - если есть заявки
  - 🔓/🔒 Изменить тип вступления
  - 👥 Изменить лимит участников
  - 🖼️ Установить аватар
  - 👢 Кикнуть игрока
- 🔙 Назад
