## 🎨 Общие рекомендации для всех промптов:

### Технические параметры:
- **Разрешение**: 1024x1024 или 1280x720
- **Стиль**: Fantasy game character art, digital illustration
- **Качество**: High quality, detailed, professional
- **Освещение**: Dramatic lighting, epic atmosphere

### Цветовые схемы по редкостям:
- **RARE (🔵)**: Синие и серебряные акценты
- **EPIC (🟣)**: Фиолетовые и золотые тона  
- **MYTHIC (🔴)**: Красные и мистические цвета

### Дополнительные теги для улучшения:
```
, fantasy RPG character, game art style, detailed illustration, 
epic pose, heroic lighting, high resolution, digital art masterpiece
```

### Негативные промпты (что исключить):
```
low quality, blurry, pixelated, cartoon style, anime, 
modern clothing, guns, cars, low resolution, bad anatomy
```

## 📝 Инструкции по использованию:

1. **Выберите основной промпт** для каждой расы
2. **Добавьте технические теги** для улучшения качества
3. **Используйте негативные промпты** для исключения нежелательных элементов
4. **Экспериментируйте** с альтернативными вариантами
5. **Сохраняйте изображения** в формате JPG в папку `images/races/`

## 🔧 Настройки для AI генераторов:

### Midjourney:
- Добавьте `--ar 1:1` для квадратного формата
- Используйте `--v 6` для лучшего качества
- Добавьте `--style raw` для более реалистичного стиля

### DALL-E / Stable Diffusion:
- Размер: 1024x1024
- Стиль: Realistic fantasy art
- Качество: Высокое

### Leonardo AI:
- Модель: Leonardo Diffusion XL
- Стиль: Fantasy Art
- Разрешение: 1024x1024
