# Система продажи предметов

## Описание
Добавлена возможность продавать предметы из инвентаря за золото.

## Как работает

### 1. Просмотр предмета
- Зайти в Инвентарь → Все предметы
- Выбрать любой предмет
- Появится кнопка "💰 Продать за Х золота"

### 2. Цены продажи
Цена продажи = 50% от базовой стоимости по редкости:

| Редкость | Базовая цена | Цена продажи |
|----------|--------------|--------------|
| ⚪️ Обычный | 50 | 25 |
| 🔵 Редкий | 150 | 75 |
| 🟣 Эпический | 400 | 200 |
| 🔴 Мистический | 1000 | 500 |
| 🟡 Легендарный | 2500 | 1250 |
| ⚫️ Секретный | 5000 | 2500 |

### 3. Процесс продажи
1. Нажать "💰 Продать за Х золота"
2. Появится подтверждение с предупреждением
3. Нажать "✅ Продать за Х💰" для подтверждения
4. Предмет удаляется, золото добавляется

## Ограничения

❌ **Нельзя продать экипированный предмет**
- Сначала нужно снять предмет
- Появится предупреждение: "❌ Сначала снимите предмет!"

⚠️ **Действие необратимо**
- После продажи предмет удаляется навсегда
- Вернуть предмет нельзя

## Изменения в коде

### bot.js

**1. Добавлена кнопка продажи в детальном виде предмета:**
```javascript
buttons.push([{ text: `💰 Продать за ${sellPrice} золота`, callback_data: `sell_item_${inventoryId}` }]);
```

**2. Обработчик показа подтверждения (`sell_item_X`):**
- Проверяет, что предмет не экипирован
- Рассчитывает цену продажи
- Показывает подтверждение с предупреждением

**3. Обработчик подтверждения продажи (`confirm_sell_X`):**
- Удаляет предмет из инвентаря
- Добавляет золото игроку
- Показывает сообщение об успешной продаже

## Формула расчета цены

```javascript
const rarityPrices = {
  'COMMON': 50,
  'RARE': 150,
  'EPIC': 400,
  'MYTHIC': 1000,
  'LEGENDARY': 2500,
  'SECRET': 5000
};
const sellPrice = Math.floor((rarityPrices[item.rarity] || 50) * 0.5);
```

## Примеры использования

### Пример 1: Продажа обычного предмета
1. Открыть инвентарь
2. Выбрать ⚪️ Обычный шлем
3. Нажать "💰 Продать за 25 золота"
4. Подтвердить продажу
5. Получить 25 золота

### Пример 2: Попытка продать экипированный предмет
1. Выбрать экипированный предмет (✅)
2. Нажать "💰 Продать"
3. Получить ошибку: "❌ Сначала снимите предмет!"
4. Нажать "❌ Снять"
5. Теперь можно продать

### Пример 3: Продажа легендарного предмета
1. Выбрать 🟡 Легендарный меч
2. Нажать "💰 Продать за 1250 золота"
3. Увидеть предупреждение: "⚠️ Это действие нельзя отменить!"
4. Подтвердить продажу
5. Получить 1250 золота

## Преимущества системы

✅ **Безопасность:** Двойное подтверждение предотвращает случайную продажу
✅ **Прозрачность:** Цена показывается до продажи
✅ **Баланс:** Цена продажи = 50% от базовой стоимости
✅ **Защита:** Нельзя продать экипированные предметы
✅ **Обратная связь:** Четкие сообщения об успехе/ошибке
