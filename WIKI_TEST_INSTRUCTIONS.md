# Инструкции по тестированию Wiki

## Проблема
Расы и предметы не отображаются на сайте, хотя файлы существуют.

## Шаги для диагностики

### 1. Откройте wiki/index.html в браузере
- Откройте файл `wiki/index.html` в браузере (Chrome, Firefox, Edge)
- Нажмите F12 чтобы открыть консоль разработчика

### 2. Проверьте консоль на ошибки
Ищите ошибки типа:
- `RACE_DATA is not defined`
- `ITEM_DATA is not defined`
- `Failed to load resource`
- CORS errors

### 3. Проверьте загрузку данных
В консоли браузера введите:
```javascript
console.log(typeof RACE_DATA);
console.log(typeof ITEM_DATA);
console.log(typeof LOCATIONS_DATA);
```

Должно вывести `"object"` для каждого. Если выводит `"undefined"`, значит файлы не загрузились.

### 4. Проверьте количество данных
```javascript
console.log(Object.keys(RACE_DATA).length); // Должно быть 23
console.log(Object.keys(ITEM_DATA).length); // Должно быть ~50+
console.log(Object.keys(LOCATIONS_DATA).length); // Должно быть 12
```

### 5. Проверьте навигацию
- Нажмите "Все расы" в меню
- Нажмите "Все предметы" в меню
- Проверьте что таблицы отображаются

## Возможные проблемы и решения

### Проблема: CORS ошибки
**Решение**: Запустите локальный веб-сервер:
```bash
cd wiki
python -m http.server 8000
# Или
npx http-server -p 8000
```
Затем откройте http://localhost:8000

### Проблема: Файлы не загружаются
**Решение**: Проверьте порядок загрузки скриптов в index.html:
```html
<script src="race_data.js"></script>
<script src="item_data.js"></script>
<script src="locations_data.js"></script>
<script src="app.js"></script>
```

### Проблема: Данные есть, но не отображаются
**Решение**: Проверьте функции showAllRaces() и showAllItems() в app.js

## Масштабирование HP мобов

На уровне леса 41:
- Базовый HP моба: 320
- Множитель: 1 + (41 * 0.1) = 5.1
- Итоговый HP: 320 * 5.1 = 1632 (примерно 1785 с учетом вариаций)
