# ⚡ Быстрый старт - Форматирование больших чисел

## Что это дает?

Вместо: `💰 Золото: 1390000`
Будет: `💰 Золото: 1.39M`

Вместо: `❤️ HP: 5000000`
Будет: `❤️ HP: 5M`

## Автоматическая установка (рекомендуется)

```bash
node apply_number_formatting.js
```

Скрипт автоматически:
- ✅ Создаст резервную копию bot.js
- ✅ Добавит импорт форматтера
- ✅ Заменит основные места с числами
- ✅ Покажет количество замен

## Ручная установка

### 1. Добавьте импорт в bot.js

В начало файла, после других `require`:

```javascript
const { smartFormat } = require('./number_formatter');
```

### 2. Замените числа на форматированные

**Было:**
```javascript
`💰 Золото: ${player.gold}\n`
```

**Стало:**
```javascript
`💰 Золото: ${smartFormat(player.gold)}\n`
```

### 3. Замените .toLocaleString()

**Было:**
```javascript
`❤️ HP: ${raid.current_hp.toLocaleString()}\n`
```

**Стало:**
```javascript
`❤️ HP: ${smartFormat(raid.current_hp)}\n`
```

## Что форматируется?

- 💰 Золото
- 💎 Кристаллы
- ✨ Опыт
- ❤️ HP (игрока и боссов)
- ⚔️ Урон
- ⚡ Сила, атака, защита
- 🏆 Награды

## Суффиксы

- **K** = 100,000+ (сто тысяч и больше)
- **M** = 1,000,000 (миллион)
- **B** = 1,000,000,000 (миллиард)
- **T** = 1,000,000,000,000 (триллион)
- **Q** = 1,000,000,000,000,000 (квадриллион)
