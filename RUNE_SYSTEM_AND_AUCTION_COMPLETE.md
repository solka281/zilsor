# Система рун и аукционов - Завершено

## Выполненные задачи

### 1. Система аукционов с уведомлениями ✅

**Автоматическое завершение аукционов:**
- Проверка каждую минуту на истекшие аукционы
- Если есть ставки: предмет передается победителю, продавец получает 80%
- Если нет ставок: аукцион помечается как expired, предмет остается у продавца

**Уведомления через Telegram:**
- `notifyBidOutbid()` - когда ставку перебили
- `notifyAuctionSold()` - когда аукцион завершен с продажей
- `notifyAuctionWon()` - когда игрок выиграл аукцион
- `notifyAuctionExpired()` - когда аукцион завершился без ставок
- `notifyItemSold()` - когда предмет куплен напрямую (не через аукцион)

**Файлы:**
- `marketplace_api.js` - добавлены функции завершения и уведомлений

### 2. Система эффектов рун ✅

**Реализованные эффекты рун:**

1. **Руна жизни** (`regen_5`)
   - Восстанавливает 5 HP в начале каждого хода
   - Применяется через `applyItemEffects()`

2. **Руна скорости** (`speed_boost_25`)
   - +25% к шансу уклонения
   - Применяется через `modifyDodgeWithItems()`

3. **Руна берсерка** (`berserk_mode`)
   - +50% к урону атакующего
   - -20% к защите (получает больше урона)
   - Применяется через `modifyDamageWithItems()`

4. **Руна защиты** (`damage_reduction_15`)
   - -15% получаемого урона
   - Применяется через `modifyDamageWithItems()`

5. **Руна мастера** (`exp_boost_30`)
   - +30% к получаемому опыту
   - Применяется через `modifyExpWithItems()`

6. **Руна бессмертия** (`immortal_chance_5`)
   - 5% шанс выжить с 1 HP при смертельном уроне
   - Применяется через `checkImmortalityRune()`

**Интеграция в боевую систему:**

**PvE бои (battle_system.js):**
- В начале хода: `applyItemEffects()` для регенерации
- При атаке игрока: `modifyDamageWithItems()` для бонусов урона
- При защите игрока: `modifyDamageWithItems()` для снижения урона
- При уклонении: `modifyDodgeWithItems()` для бонуса к dodge
- При смерти: `checkImmortalityRune()` для шанса выжить
- После победы: `modifyExpWithItems()` для бонуса опыта

**PvP дуэли (duels.js):**
- Эффекты предметов уже включены в `itemEffects` массив
- Система готова к использованию рун в дуэлях

**Файлы:**
- `race_abilities.js` - добавлены функции эффектов рун
- `battle_system.js` - интегрированы эффекты в PvE бои
- `bot.js` - добавлен модификатор опыта от рун

## Как работает система

### Аукционы
1. Игрок создает аукцион через маркетплейс
2. Другие игроки делают ставки
3. Каждая новая ставка уведомляет предыдущего лидера
4. Через указанное время аукцион автоматически завершается
5. Победитель получает предмет, продавец получает 80% от ставки
6. Все участники получают уведомления в Telegram

### Руны
1. Игрок экипирует руну в слот "🔮 Руна" (artifact_2)
2. Эффект руны автоматически применяется в бою:
   - Регенерация - в начале каждого хода
   - Урон/защита - при расчете урона
   - Уклонение - при проверке dodge
   - Бессмертие - при проверке смерти
   - Опыт - при получении награды
3. Эффекты складываются с расовыми способностями

## Технические детали

**Структура эффектов:**
- Эффекты хранятся в поле `special_effect` предмета
- Формат: `effect_name` или `effect_name_value`
- Примеры: `regen_5`, `speed_boost_25`, `berserk_mode`

**Порядок применения:**
1. Расовые способности
2. Эффекты рун/предметов
3. Базовые модификаторы (крит, уклонение)
4. Случайный разброс урона

**Экспортированные функции из race_abilities.js:**
- `applyItemEffects()` - эффекты в начале хода
- `modifyDamageWithItems()` - модификация урона
- `modifyDodgeWithItems()` - модификация уклонения
- `modifyExpWithItems()` - модификация опыта
- `checkImmortalityRune()` - проверка бессмертия
