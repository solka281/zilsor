# Новый Рейд-Босс: Владыка тьмы

## Характеристики босса

- **Имя**: Владыка тьмы
- **Уровень**: 2
- **Начальное HP**: 175,000
- **Изображение**: `dark_lord.jpg` (уже готово)
- **Описание**: "Повелитель темных сил и теней"
- **Кулдаун**: 5 часов (между рейдами)

## Прогрессия HP
- **Базовое HP**: 175,000
- **Надбавка за каждую победу**: +5,000 HP
- **Формула**: `Текущее HP = 175,000 + (количество побед × 5,000)`

## Награды

### Базовые награды (делятся пропорционально урону):
- **Золото**: 2,000 (общий пул)
- **Алмазы**: 30 (общий пул)
- **Опыт**: 4,000 (общий пул)

### Специальные награды:
- **Топ-1 по урону**: Гарантированно получает **Амулет тьмы**
  - Редкость: MYTHIC
  - Элемент тьмы: 🌑
  - Бонусы: +80 силы, +50 HP, +40 атаки, +20 защиты

## Условия участия

### Обязательные требования:
1. **Уровень расы**: 10+ (минимум)
2. **Опыт рейдов**: Хотя бы одно участие в рейде "Повелитель ветра"

### Проверка условий:
- Система автоматически проверяет условия при попытке присоединения
- Если условия не выполнены, игрок получает сообщение с объяснением
- Кнопка "Присоединиться" заменяется на "❌ Недоступно"

## Технические особенности

### Обновления базы данных:
- Добавлены колонки в таблицу `raid_bosses`:
  - `cooldown_hours` - кулдаун между рейдами
  - `requirements` - JSON с условиями участия
  - `special_rewards` - JSON со специальными наградами

### Новые функции:
- `checkRaidRequirements()` - проверка условий участия
- Обновленная `completeRaid()` - выдача специальных наград
- Обновленная `getNextRaidTime()` - учет кулдауна босса

### Интерфейс:
- Отображение условий участия в меню босса
- Показ специальных наград в описании
- Уведомления о специальных наградах в результатах

## Пример условий участия (JSON):
```json
{
  "min_race_level": 10,
  "required_raid_participation": "Повелитель ветра"
}
```

## Пример специальных наград (JSON):
```json
{
  "top_1_guaranteed_item": "Амулет тьмы"
}
```

## Игровой баланс

### Сложность:
- **HP в 2.3 раза больше** чем у Повелителя ветра
- **Более высокие требования** для участия
- **Длительный кулдаун** (5 часов vs 2 часа)

### Награды:
- **В 2 раза больше золота** (2000 vs 1000)
- **В 1.5 раза больше алмазов** (30 vs 20)
- **В 4 раза больше опыта** (4000 vs 1000)
- **Уникальная награда** для лучшего игрока

## Автоматическое создание
- Босс автоматически создается при инициализации системы рейдов
- Если босс уже существует, создание пропускается
- Все данные сохраняются в базе данных

## Совместимость
- Полностью совместимо с существующей системой рейдов
- Старые рейды продолжают работать без изменений
- Новые функции работают только с новыми боссами
