# Исправление багов ежедневных квестов

## Проблемы
1. ❌ Квесты не выполняются (прогресс не обновляется)
2. ❌ У одного игрока 1 квест, у другого 6 квестов (должно быть 3)
3. ❌ Выполненные квесты не засчитываются

## Причины

### 1. Асинхронные UPDATE без ожидания
```javascript
// БЫЛО (неправильно):
db.run(`UPDATE ...`); // Запускается
callback(null, completedQuests); // Вызывается сразу, не дожидаясь UPDATE
```

### 2. Неправильная проверка количества квестов
```javascript
// БЫЛО:
if (result.count >= 3) return; // Всегда выдавало 3 новых
// Не учитывало уже существующие квесты
```

### 3. Старые квесты не удалялись
- Квесты с прошлых дней оставались в базе
- Накапливались и показывались вместе с новыми

## Решения

### 1. Исправлена асинхронность в updateQuestProgress
```javascript
let processed = 0;

quests.forEach(quest => {
  db.run(`UPDATE ...`, (err) => {
    processed++;
    if (processed === quests.length) {
      callback(null, completedQuests); // Вызываем только после всех UPDATE
    }
  });
});
```

**Результат:** Квесты теперь корректно обновляются и выполняются

### 2. Исправлена выдача квестов
```javascript
// Вычисляем сколько квестов нужно выдать
const questsToAssign = 3 - (result ? result.count : 0);

// Получаем только нужное количество
db.all(`SELECT * FROM daily_quests_pool ORDER BY RANDOM() LIMIT ?`, [questsToAssign])
```

**Результат:** Всегда выдается ровно 3 квеста (или дополняется до 3)

### 3. Добавлена очистка старых квестов
```javascript
// Удаляем квесты не с сегодняшней даты
db.run(`DELETE FROM player_daily_quests 
        WHERE player_id = ? AND DATE(assigned_at) != ?`, [playerId, today])
```

**Результат:** Старые квесты автоматически удаляются

### 4. Добавлено подробное логирование
```javascript
console.log(`📊 Обновление прогресса: playerId=${playerId}, type=${questType}`);
console.log(`Найдено ${quests.length} квестов типа ${questType}`);
console.log(`Квест "${quest.quest_name}": ${quest.progress} -> ${newProgress}`);
console.log(`✅ Квест "${quest.quest_name}" выполнен!`);
```

**Результат:** Легко отслеживать работу системы в логах

## Тестирование

### Проверка выдачи квестов
```
1. Игрок открывает меню квестов
2. Должно появиться ровно 3 квеста
3. При повторном открытии - те же 3 квеста
4. На следующий день - новые 3 квеста
```

### Проверка выполнения
```
1. Игрок получает квест "Победите в 1 дуэли"
2. Побеждает в дуэли
3. Должно прийти сообщение "✅ Квест выполнен!"
4. Награда должна быть выдана
5. В меню квестов должно остаться 2 квеста
```

### Проверка прогресса
```
1. Игрок получает квест "Победите в 3 дуэлях"
2. Побеждает в 1 дуэли
3. Прогресс: 1/3
4. Побеждает еще в 1 дуэли
5. Прогресс: 2/3
6. Побеждает в 3-й дуэли
7. Квест выполнен! Награда выдана
```
