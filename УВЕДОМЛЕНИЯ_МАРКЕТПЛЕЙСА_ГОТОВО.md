# ✅ Уведомления маркетплейса - ИСПРАВЛЕНО

## Проблемы которые были решены:

### 1. 🔔 Уведомления о покупках не приходили
**Причина:** Циклическая зависимость между `bot.js` и `marketplace_api.js`

**Решение:** 
- Установлена глобальная переменная `global.telegramBot`
- Добавлен резервный способ через прямой импорт
- Добавлен экспорт бота из `bot.js`

### 2. ❌ Ошибка в аукционах
**Причина:** Неправильный статус `'expired'` не разрешен в CHECK constraint

**Решение:** Заменен на `'cancelled'`

## Технические детали:

### Двойной подход к получению бота:
```javascript
// Способ 1: Глобальная переменная (основной)
if (global.telegramBot) {
  bot = global.telegramBot;
}
// Способ 2: Прямой импорт (резервный)
else {
  const botModule = require('./bot');
  bot = botModule.bot;
}
```

### Экспорт бота:
```javascript
// В bot.js
module.exports = { bot };
```

### Исправление аукционов:
```javascript
// Было: status = 'expired' ❌
// Стало: status = 'cancelled' ✅
```
