## 🎯 ЧТО СДЕЛАНО:

### ✅ 1. Заменена кнопка "Темный лес" на "Прохождение"
- Новая кнопка: `🗺️ Прохождение` вместо `🌲 Темный лес`
- Callback: `adventure_menu` вместо `dark_forest`

### ✅ 2. Создано меню выбора локаций
- Показывает все доступные локации до текущего уровня игрока
- Кнопка быстрого доступа к текущему уровню: `⚡ Текущий уровень (X)`
- Локации отображаются с диапазонами уровней: `Болотистая Топь (1-10)`

### ✅ 3. Создано меню выбора уровней в локации
- Показывает все доступные уровни в выбранной локации
- Боссы отмечены короной: `👑20` вместо `20`
- Уровни отображаются по 5 в ряд для удобства

### ✅ 4. Добавлена таблица убитых боссов
```sql
CREATE TABLE killed_bosses (
  user_id INTEGER,
  boss_level INTEGER,
  killed_at INTEGER,
  PRIMARY KEY (user_id, boss_level)
)
```

### ✅ 5. Логика повторного прохождения боссов
- Убитые боссы помечаются как "(Повтор)"
- **НЕ дают кристаллы** при повторном убийстве
- Дают обычные награды (золото, опыт, предметы)

### ✅ 6. Система прогрессии уровня леса
- Уровень леса повышается **ТОЛЬКО** при прохождении текущего максимального уровня
- Возврат к старым уровням **НЕ повышает** уровень леса
- Сохраняется прогресс игрока

### ✅ 7. Функции кулдауна
- `getCooldown(userId, actionType)` - проверка кулдауна
- `setCooldown(userId, actionType, seconds)` - установка кулдауна
- Поддержка типов: `forest`, `duel`, `work`

### ✅ 8. Функция начала боя в приключении
- `startAdventureBattle(userId, player, enemy, selectedLevel, isBossRerun)`
- Создает бой с учетом выбранного уровня
- Отображает информацию о локации и враге

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ:

### Новые обработчики callback:
- `adventure_menu` - главное меню прохождения
- `adventure_location_X` - выбор локации (X = ID локации)
- `adventure_level_X` - выбор уровня (X = номер уровня)

### Изменения в handleBattleEnd:
```javascript
// Повышаем уровень леса ТОЛЬКО если это текущий максимальный уровень
let forestLevelUp = 0;
if (victory && battle.selectedLevel) {
  if (battle.selectedLevel === (player.forest_level || 1)) {
    forestLevelUp = 1;
  }
}

// Записываем убитого босса
if (victory && enemy.isBoss && !battle.isBossRerun) {
  db.run(`INSERT OR REPLACE INTO killed_bosses (user_id, boss_level, killed_at) VALUES (?, ?, ?)`,
    [userId, battle.selectedLevel, now]);
}
```

### Проверка убитых боссов:
```javascript
db.get(`SELECT * FROM killed_bosses WHERE user_id = ? AND boss_level = ?`, 
  [userId, selectedLevel], (err, killedBoss) => {
  
  if (killedBoss) {
    // Босс уже убит - без кристаллов
    enemy.crystalReward = 0;
    enemy.name += ' (Повтор)';
    isBossRerun = true;
  }
});
```

## 🎯 ПРЕИМУЩЕСТВА НОВОЙ СИСТЕМЫ:

### ✅ Для игроков:
- **Гибкость:** Можно возвращаться к любым пройденным уровням
- **Фарм:** Можно фармить золото/опыт/предметы на старых уровнях
- **Прогресс:** Уровень леса не откатывается назад
- **Ясность:** Четко видно какие локации и уровни доступны

### ✅ Для баланса:
- **Контроль прогресса:** Нельзя "проскочить" уровни
- **Ограничение кристаллов:** Боссы дают кристаллы только один раз
- **Мотивация:** Есть смысл проходить новые уровни для прогресса

### ✅ Для UX:
- **Интуитивность:** Понятная навигация по локациям
- **Быстрый доступ:** Кнопка текущего уровня
- **Визуальные подсказки:** Короны для боссов, диапазоны уровней
