# Система сообщений от администрации

## 🎯 Добавлено:

### 📢 Глобальные сообщения
Админ может отправить сообщение **всем игрокам** одновременно.

**Как использовать:**
1. Открыть админ-панель `/admin`
2. Нажать кнопку "📢 Глобальное сообщение"
3. Ввести текст сообщения
4. Сообщение будет отправлено всем игрокам в базе данных

**Формат сообщения:**
```
📢 ОБЪЯВЛЕНИЕ ОТ АДМИНИСТРАЦИИ

[Ваш текст]
```

**Особенности:**
- ✅ Поддерживается Markdown форматирование
- ✅ Задержка 100мс между сообщениями (защита от спама)
- ✅ Статистика отправки (успешно/ошибки)
- ✅ Автоматический подсчет игроков

### 💬 Личные сообщения
Админ может отправить сообщение **конкретному игроку**.

**Как использовать:**
1. Открыть админ-панель `/admin`
2. Нажать кнопку "💬 Личное сообщение"
3. Ввести ID игрока
4. Ввести текст сообщения
5. Сообщение будет отправлено выбранному игроку

**Формат сообщения:**
```
💬 СООБЩЕНИЕ ОТ АДМИНИСТРАЦИИ

[Ваш текст]
```

**Особенности:**
- ✅ Поддерживается Markdown форматирование
- ✅ Проверка существования игрока
- ✅ Показывается имя игрока перед отправкой
- ✅ Подтверждение успешной отправки

## 🎮 Примеры использования:

### Глобальное объявление:
```
📢 Внимание!

Сегодня в 20:00 будет проведено техническое обслуживание.
Бот будет недоступен около 30 минут.

Приносим извинения за неудобства! 🛠️
```

### Личное сообщение:
```
💬 Здравствуйте!

Спасибо за ваш отзыв о боте.
Мы учли ваши пожелания и добавим новые функции в следующем обновлении.

С уважением, команда разработки 🎮
```

## 🔧 Технические детали:

### Глобальная рассылка:
```javascript
// Получаем всех игроков
db.all(`SELECT user_id FROM players`, (err, players) => {
  // Отправляем с задержкой 100мс между сообщениями
  players.forEach((player, index) => {
    setTimeout(() => {
      bot.sendMessage(player.user_id, message);
    }, index * 100);
  });
});
```

### Личное сообщение:
```javascript
// Проверяем игрока
db.get(`SELECT user_id, username FROM players WHERE user_id = ?`, [targetUserId], (err, player) => {
  // Отправляем сообщение
  bot.sendMessage(targetUserId, message);
});
```

## 📊 Статистика рассылки:

После глобальной рассылки админ получает отчет:
```
✅ Рассылка завершена!

📤 Отправлено: 245
❌ Ошибок: 5
👥 Всего игроков: 250
```

## 🛡️ Безопасность:

- ✅ Доступ только для админа (проверка `config.ADMIN_USERNAME`)
- ✅ Валидация ID игрока
- ✅ Проверка существования игрока
- ✅ Обработка ошибок отправки
- ✅ Защита от пустых сообщений

## ✅ Статус:
**ДОБАВЛЕНО!** Админ может отправлять глобальные и личные сообщения от имени бота.
