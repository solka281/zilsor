### Как работает:

1. **Игрок нажимает "⚔️ Дуэль"**
   - Начинается поиск противника
   - Показывается сообщение с кнопкой "❌ Отменить поиск"

2. **Поиск работает бесконечно:**
   - Каждую секунду проверяется, найден ли противник
   - Если найден - начинается бой
   - Если игрок отменил - поиск останавливается

3. **Отмена поиска:**
   - Игрок нажимает "❌ Отменить поиск"
   - Показывается "❌ Поиск отменен"
   - Возврат в главное меню

### Технические детали:

**Удалено:**
```javascript
let pollCount = 0;
const maxPolls = 30; // 30 секунд

if (pollCount >= maxPolls) {
  // Таймаут поиска
  clearInterval(pollInterval);
  db.run(`UPDATE players SET is_searching_duel = 0...`);
  bot.sendMessage(userId, '⏰ Поиск завершен...');
}
```

**Добавлено:**
```javascript
// Проверяем, не отменил ли игрок поиск
db.get(`SELECT is_searching_duel FROM players WHERE user_id = ?`, [userId], (err, player) => {
  if (err || !player || player.is_searching_duel === 0) {
    // Игрок отменил поиск
    clearInterval(pollInterval);
    console.log(`❌ Игрок ${userId} отменил поиск`);
  }
});
```
