# Система выбора рейд-боссов со скроллом

## Новая функциональность

Добавлена система выбора рейд-босса с навигацией, аналогичная просмотру рас.

## Особенности системы

### 1. Скролл-навигация
- Кнопки "◀️" и "▶️" для перехода между боссами
- Показ текущей позиции: "ВЫБОР БОССА (1/2)"
- Плавная навигация между доступными боссами

### 2. Детальная информация о боссе
- **Название и уровень** босса
- **Описание** и лор босса
- **HP с прогрессией** (базовое + бонус за победы)
- **Количество побед** и бонус HP
- **Кулдаун** между рейдами
- **Условия участия** (если есть)
- **Специальные награды** (если есть)
- **Стандартные награды** (золото, алмазы, опыт)

### 3. Проверка условий участия
- Автоматическая проверка требований для каждого босса
- Отображение причины недоступности
- Блокировка кнопки создания рейда при невыполнении условий

### 4. Умное отображение состояния
- **Активный рейд**: показ текущего рейда с кнопкой атаки
- **Ожидание**: показ времени до следующего рейда
- **Выбор**: навигация по доступным боссам

## Новые функции

### `showRaidBossSelection(chatId, messageId, currentIndex, userId)`
Основная функция отображения выбора босса:
- Проверяет наличие активного рейда
- Показывает время ожидания или выбор боссов
- Обрабатывает навигацию и условия участия

### Обработчики callback_data:
- `raid_boss_page_{index}` - навигация по боссам
- `create_raid_{bossId}` - создание рейда с выбранным боссом

### Кнопки:
- `◀️` `▶️` - навигация (если есть другие боссы)
- `⚔️ Создать рейд с этим боссом` - создание рейда
- `❌ Требуется участие в рейде "Повелитель ветра"` - заблокировано
- `🔙 Назад` - возврат в меню рейдов

## Логика работы

### 1. Проверка активного рейда
```javascript
raids.getActiveRaid((err, activeRaid) => {
  if (activeRaid) {
    // Показываем активный рейд
    return showActiveRaid();
  }
  // Показываем выбор боссов
});
```

### 2. Проверка времени рейда
```javascript
raids.getNextRaidTime((err, nextRaid) => {
  if (!nextRaid.ready) {
    // Показываем время ожидания
    return showWaitingTime();
  }
  // Показываем выбор боссов
});
```

### 3. Проверка условий участия
```javascript
raids.checkRaidRequirements(userId, boss.name, (err, requirements) => {
  if (!requirements.canJoin) {
    // Блокируем кнопку создания
    button = { text: `❌ ${requirements.reason}`, callback_data: 'noop' };
  }
});
```

## Преимущества новой системы

1. **Выбор босса**: игроки могут выбирать из доступных боссов
2. **Информативность**: полная информация о боссе перед созданием рейда
3. **Условия участия**: четкое отображение требований
4. **Навигация**: удобный скролл между боссами
5. **Состояние системы**: понятное отображение текущего состояния рейдов

## Совместимость

- Полностью совместимо с существующей системой рейдов
- Сохраняет все функции админ-панели
- Работает с любым количеством боссов
- Автоматически адаптируется к новым боссам
