# Исправление использования зелий в бою

## Проблема
Зелья не работали в бою - появлялось сообщение "⚔️ Завершите текущий бой!"

## Причина
В обработчике `callback_query` была проверка активного боя, которая блокировала все действия кроме:
- `battle_attack`
- `battle_defend`
- `battle_special`

Действие `battle_use_potion` не было в списке разрешенных, поэтому блокировалось.

## Решение
Добавлено `battle_use_potion` в список разрешенных действий во время боя:

```javascript
const battleActions = ['battle_attack', 'battle_defend', 'battle_special', 'battle_use_potion'];
```

## Изменения

### bot.js (строка ~2606)
**До:**
```javascript
const battleActions = ['battle_attack', 'battle_defend', 'battle_special'];
```

**После:**
```javascript
const battleActions = ['battle_attack', 'battle_defend', 'battle_special', 'battle_use_potion'];
```

## Дополнительное изменение

### config.js
Изменен кулдаун дуэлей с 30 минут на 10 минут:

**До:**
```javascript
DUEL_COOLDOWN: 1800, // 30 минут
```

**После:**
```javascript
DUEL_COOLDOWN: 600, // 10 минут
```

## Результат

✅ Зелья теперь можно использовать в бою
✅ Кнопка "🧪 [Название] (X шт)" работает корректно
✅ HP восстанавливается
✅ Количество зелий уменьшается
✅ Кулдаун дуэлей теперь 10 минут
