# Система способностей рас

## Дата: 2026-04-22

## Обзор

Создана полноценная система способностей рас с уникальными механиками для каждой расы. Способности делятся на несколько типов:
- **Атакующие** - увеличивают урон или добавляют эффекты
- **Защитные** - снижают получаемый урон
- **Лечащие** - восстанавливают HP
- **Пассивные** - постоянные бонусы к характеристикам
- **Уникальные** - специальные механики (возрождение, неуязвимость)

## Реализованные способности

### Обычные расы (COMMON)

#### 🧑 Человек - Адаптация
- **Эффект**: +5% к получению опыта
- **Тип**: Пассивная
- **Применение**: При получении опыта после боя

#### 🧝 Эльф - Меткость
- **Эффект**: +10% к атаке
- **Шанс**: 25%
- **Тип**: Атакующая
- **Сообщение**: "🎯 Меткий выстрел!"

#### 🧔 Дварф - Стойкость
- **Эффект**: +15% к защите (снижение урона)
- **Шанс**: 30%
- **Тип**: Защитная
- **Сообщение**: "🛡️ Стойкость дварфа! -X урона"

#### 👹 Орк - Ярость
- **Эффект**: +15% к атаке при HP < 30%
- **Условие**: Здоровье ниже 30%
- **Тип**: Атакующая
- **Сообщение**: "😡 Ярость орка! Урон увеличен!"

### Редкие расы (RARE)

#### 🌑 Темный Эльф - Теневой удар
- **Эффект**: +50% урона (критический удар)
- **Шанс**: 20%
- **Тип**: Атакующая
- **Сообщение**: "🌑 Теневой удар!"

#### 💪 Полуорк - Гибридная сила
- **Эффект**: +10% ко всем характеристикам
- **Тип**: Пассивная
- **Применение**: При расчете статов игрока

#### 🔧 Гном - Инженерия
- **Эффект**: +20% к эффективности предметов
- **Тип**: Пассивная
- **Применение**: При расчете бонусов от экипировки

### Эпические расы (EPIC)

#### 🐉 Драконорожденный - Драконье дыхание
- **Эффект**: +80% урона
- **Шанс**: 15%
- **Тип**: Атакующая
- **Сообщение**: "🐉 Драконье дыхание!"

#### 😈 Демон - Адское пламя
- **Эффект**: Урон со временем (30% от урона атаки в течение 2 ходов)
- **Шанс**: 20%
- **Тип**: Атакующая + DoT (Damage over Time)
- **Сообщение**: "🔥 Адское пламя! Противник горит (X урона/ход)"

#### 😇 Ангел - Божественная защита
- **Эффект**: Восстановление 15% HP
- **Шанс**: 20%
- **Тип**: Лечащая
- **Сообщение**: "✨ Божественная защита! +X HP"

### Мистические расы (MYTHIC)

#### 🔥 Феникс - Возрождение
- **Эффект**: Воскрешение с 50% HP после смерти (1 раз за бой)
- **Условие**: Смерть в бою
- **Тип**: Уникальная
- **Сообщение**: "🔥 ВОЗРОЖДЕНИЕ! Феникс восстал из пепла с X HP!"

#### 🧛 Вампир - Кровопийство
- **Эффект**: Восстановление 25% от нанесенного урона
- **Шанс**: 100% (всегда активна)
- **Тип**: Лечащая
- **Сообщение**: "🩸 Кровопийство! +X HP"

### Легендарные расы (LEGENDARY)

#### ⚡ Титан - Титаническая мощь
- **Эффект**: Урон x2 (удваивает урон атаки)
- **Шанс**: 10%
- **Тип**: Атакующая
- **Сообщение**: "⚡ ТИТАНИЧЕСКАЯ МОЩЬ! Урон удвоен!"

#### ⚔️ Бог Войны - Божественная ярость
- **Эффект**: Неуязвимость на 2 хода (блокирует весь урон)
- **Шанс**: 8%
- **Тип**: Защитная
- **Сообщение**: "⚔️ БОЖЕСТВЕННАЯ ЯРОСТЬ! Неуязвимость на 2 хода!"

### Секретные расы (SECRET)

#### 🌌 Древний - Первородная сила
- **Эффект**: Урон x2.5 + восстановление 20% HP
- **Шанс**: 15%
- **Тип**: Атакующая + Лечащая
- **Сообщение**: "🌌 ПЕРВОРОДНАЯ СИЛА! Урон x2.5 и +X HP!"

## Технические детали

### Файл: race_abilities.js

Модуль содержит следующие функции:

#### applyRaceAbility(attacker, defender, attackResult, battleContext)
Применяет атакующие и лечащие способности расы после атаки.

**Параметры:**
- `attacker` - объект атакующего с полями: specialAbility, currentHP, maxHP
- `defender` - объект защищающегося
- `attackResult` - результат атаки с полями: damage, critical, special
- `battleContext` - контекст боя для отслеживания эффектов

**Возвращает:** Модифицированный attackResult с добавленным полем abilityMessage

#### applyDefensiveAbility(defender, incomingDamage, battleContext)
Применяет защитные способности при получении урона.

**Параметры:**
- `defender` - объект защищающегося
- `incomingDamage` - входящий урон
- `battleContext` - контекст боя

**Возвращает:** Объект { damage, abilityMessage }

#### applyTurnEffects(target, battleContext)
Применяет эффекты в начале хода (горение, яды).

**Параметры:**
- `target` - цель эффектов
- `battleContext` - контекст боя с активными эффектами

**Возвращает:** Массив сообщений об эффектах

#### checkPhoenixRevive(player, battleContext)
Проверяет и активирует возрождение Феникса.

**Параметры:**
- `player` - игрок с способностью Феникса
- `battleContext` - контекст боя

**Возвращает:** true если возрождение произошло, false иначе

#### applyExpBonus(player, baseExp)
Применяет бонусы к опыту (Адаптация человека).

**Параметры:**
- `player` - игрок
- `baseExp` - базовый опыт

**Возвращает:** Модифицированный опыт

#### applyStatBonus(player)
Применяет бонусы к характеристикам (Гибридная сила полуорка).

**Параметры:**
- `player` - объект игрока

**Возвращает:** Модифицированный объект игрока

#### applyItemBonus(player, itemBonus)
Применяет бонусы к предметам (Инженерия гнома).

**Параметры:**
- `player` - игрок
- `itemBonus` - бонус от предмета

**Возвращает:** Модифицированный бонус

## Интеграция с боевыми системами

### Требуется интеграция в:

1. **bot.js** - PvE бои (лес)
   - Импортировать race_abilities.js
   - Вызывать applyRaceAbility после каждой атаки
   - Вызывать applyDefensiveAbility при получении урона
   - Вызывать applyTurnEffects в начале каждого хода
   - Проверять checkPhoenixRevive при смерти игрока

2. **duels.js** - PvP дуэли
   - Аналогично PvE боям
   - Обе стороны могут использовать способности

3. **battle_system.js** - Общая боевая система
   - Интегрировать во все расчеты урона

4. **bot.js** - Получение опыта
   - Использовать applyExpBonus при начислении опыта

5. **duels.js / bot.js** - Расчет силы игрока
   - Использовать applyStatBonus при расчете характеристик
   - Использовать applyItemBonus при расчете бонусов от предметов

## Контекст боя (battleContext)

Объект для отслеживания состояния боя:

```javascript
{
  phoenixRevived: false,      // Феникс уже возрождался?
  godShield: 0,               // Ходов неуязвимости осталось
  burnDamage: 0,              // Урон от горения
  burnTurns: 0                // Ходов горения осталось
}
```

## Баланс

Способности сбалансированы по редкости:
- **COMMON**: Слабые эффекты (5-15% бонусы)
- **RARE**: Средние эффекты (10-20% бонусы)
- **EPIC**: Сильные эффекты (15-80% бонусы, лечение)
- **MYTHIC**: Очень сильные (возрождение, вампиризм)
- **LEGENDARY**: Экстремальные (удвоение урона, неуязвимость)
- **SECRET**: Божественные (x2.5 урон + лечение)

3. Интегрировать в battle_system.js
4. Добавить визуальные эффекты для способностей
5. Протестировать баланс всех способностей
6. Добавить статистику использования способностей

## Примечания

- Все способности работают автоматически
- Игрок видит сообщения о срабатывании способностей
- Способности учитывают баланс игры
- Редкие расы имеют более мощные способности
