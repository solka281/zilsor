# Marketplace Currency System Update

## Изменения в системе маркетплейса

### ✅ Выполненные задачи:

1. **Удалена функция обмена валют**
   - Убраны эндпоинты `/exchange/buy-crystals` и `/exchange/sell-crystals`
   - Удалены функции `buyCrystals()` и старая `sellCrystals()` из JavaScript

2. **Добавлена возможность продажи валют**
   - Игроки могут выставлять на продажу золото и кристаллы
   - Золото продается блоками по 1000 (цена за 1000 золота в кристаллах)
   - Кристаллы продаются поштучно (цена за 1 кристалл в золоте)

3. **Реализована система комиссии 20%**
   - При покупке предметов продавец получает 80% от цены
   - При покупке валют продавец получает 80% от цены
   - Комиссия 20% удерживается системой

4. **Обновлен интерфейс**
   - Вкладка "Обмен" переименована в "Валюты"
   - Добавлены формы для продажи золота и кристаллов
   - Добавлены карточки валют с отображением курса и продавца

### 🗄️ Структура базы данных:

**Новая таблица `marketplace_currencies`:**
```sql
CREATE TABLE marketplace_currencies (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    seller_id INTEGER NOT NULL,
    seller_name TEXT NOT NULL,
    currency_type TEXT NOT NULL CHECK(currency_type IN ('gold', 'crystals')),
    amount INTEGER NOT NULL,
    price_per_unit INTEGER NOT NULL,
    total_price INTEGER NOT NULL,
    price_currency TEXT NOT NULL CHECK(price_currency IN ('gold', 'crystals')),
    rate_text TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 🔧 API эндпоинты:

1. **GET `/api/marketplace/currencies`** - получить валюты на продаже
2. **POST `/api/marketplace/sell-currency`** - выставить валюту на продажу
   - Параметры: `userId`, `currencyType`, `amount`, `pricePerUnit`
3. **POST `/api/marketplace/buy-currency`** - купить валюту
   - Параметры: `userId`, `currencyId`

### 💰 Система комиссии:

- **Предметы**: Покупатель платит 100%, продавец получает 80%
- **Валюты**: Покупатель платит 100%, продавец получает 80%
- **Комиссия**: 20% удерживается системой

3. **Покупка валют**:
