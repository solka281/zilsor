# Кулдаун темного леса изменен на 2 минуты

## ✅ Что изменено:

### Кулдаун темного леса

**Было:**
- ⏱️ Кулдаун: 5 минут (300 секунд)
- Игрок мог заходить в темный лес раз в 5 минут

**Стало:**
- ⏱️ Кулдаун: 2 минуты (120 секунд)
- Игрок может заходить в темный лес раз в 2 минуты

### Преимущества:

✅ Более частый фарм золота и опыта
✅ Быстрее прокачка уровня леса
✅ Больше активности в игре
✅ Меньше ожидания между боями

### Технические детали:

**Изменено в bot.js:**
```javascript
// БЫЛО:
const cooldown = checkCooldown(player.last_forest_time, 300); // 5 минут

// СТАЛО:
const cooldown = checkCooldown(player.last_forest_time, 120); // 2 минуты
```

### Как работает:

1. Игрок заходит в темный лес
2. Побеждает врага или проигрывает
3. Устанавливается кулдаун 2 минуты
4. Через 2 минуты можно снова зайти

### Сообщение о кулдауне:

Если игрок пытается зайти раньше:
```
❌ Темный лес недоступно!
⏰ Можно использовать через X минут Y секунд
```
