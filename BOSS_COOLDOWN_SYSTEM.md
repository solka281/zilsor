# ⏰ Система кулдаунов боссов

## 🎯 НОВАЯ МЕХАНИКА:

### Разные кулдауны для разных типов врагов:
- **🐺 Обычные мобы:** 10 секунд
- **👑 Боссы:** 10 минут (600 секунд)

## 🗄️ БАЗА ДАННЫХ:

### Новая таблица `boss_cooldowns`:
```sql
CREATE TABLE boss_cooldowns (
  user_id INTEGER,
  boss_level INTEGER,
  last_attempt INTEGER,
  PRIMARY KEY (user_id, boss_level)
)
```

**Отслеживает:** Когда каждый игрок последний раз атаковал каждого босса.

## 🔧 НОВЫЕ ФУНКЦИИ:

### 1. `getBossCooldown(userId, bossLevel)`
```javascript
// Проверяет кулдаун конкретного босса для игрока
const cooldown = await getBossCooldown(userId, 20); // Босс уровня 20
if (cooldown.ready) {
  // Можно атаковать
} else {
  // Показать время до готовности: cooldown.remaining
}
```

### 2. `setBossCooldown(userId, bossLevel)`
```javascript
// Устанавливает кулдаун после атаки на босса
setBossCooldown(userId, 20); // Босс уровня 20 на кулдауне
```

## 🎮 КАК РАБОТАЕТ:

### При выборе уровня:
1. **Проверяется тип врага:** босс или обычный моб
2. **Для босса:** проверяется `getBossCooldown(userId, bossLevel)`
3. **Для моба:** проверяется обычный `getCooldown(userId, 'forest')`

### При атаке:
1. **На босса:** устанавливается `setBossCooldown(userId, bossLevel)` (10 минут)
2. **На моба:** устанавливается `setCooldown(userId, 'forest')` (10 секунд)

## 🖼️ ВИЗУАЛЬНЫЕ ИНДИКАТОРЫ:

### В меню выбора уровней:
- **👑20** - Босс доступен для атаки
- **⏰20** - Босс на кулдауне (показывает время)
- **15** - Обычный моб (всегда доступен через 10 сек)

### Пояснение в интерфейсе:
```
👑 - Босс доступен | ⏰ - Босс на кулдауне (10 мин)
```

## 📊 ПРИМЕРЫ РАБОТЫ:

### Сценарий 1: Атака на босса
```
Игрок → Уровень 20 (Альфа Волк)
Система → Проверяет getBossCooldown(userId, 20)
Результат → Если готов: бой, если нет: "Босс уровня 20 недоступен! ⏰ Можно использовать через 7м 23с"
```

### Сценарий 2: Атака на моба
```
Игрок → Уровень 15 (Лесной Волк)
Система → Проверяет getCooldown(userId, 'forest')
Результат → Если готов: бой, если нет: "Прохождение недоступно! ⏰ Можно использовать через 3с"
```

### Сценарий 3: Разные боссы
```
Игрок атакует босса уровня 20 → кулдаун 10 минут на уровень 20
Игрок может сразу атаковать босса уровня 30 → у него свой кулдаун
```

## 🎯 ПРЕИМУЩЕСТВА:

### ✅ Для баланса:
- **Боссы не фармятся** слишком часто
- **Обычные мобы** остаются доступными для фарма
- **Каждый босс** имеет индивидуальный кулдаун

### ✅ Для игроков:
- **Ясность:** видно какие боссы доступны
- **Планирование:** можно планировать атаки на разных боссов
- **Прогресс:** не блокирует прохождение новых уровней

### ✅ Для геймплея:
- **Мотивация** проходить новые уровни
- **Разнообразие** в выборе активностей
- **Стратегия** в планировании времени

- ✅ Индивидуальные кулдауны для каждого босса
