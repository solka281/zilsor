# Исправление уведомлений маркетплейса

## Проблема:
Не приходили уведомления когда кто-то покупает предметы в маркетплейсе.

## Причина:
Циклическая зависимость между `bot.js` и `marketplace_api.js` при использовании `require('./bot')` в функциях уведомлений.

## Решение:

### ✅ Установлена глобальная переменная бота
В `bot.js` добавлено:
```javascript
const bot = new TelegramBot(config.BOT_TOKEN, { polling: true });

// Устанавливаем глобальную переменную для доступа из других модулей
global.telegramBot = bot;
```

### ✅ Исправлены все функции уведомлений в `marketplace_api.js`:
- `notifyItemSold()` - уведомление о покупке предмета
- `notifyBidOutbid()` - уведомление о перебитой ставке
- `notifyAuctionSold()` - уведомление о продаже на аукционе
- `notifyAuctionWon()` - уведомление о выигрыше аукциона
- `notifyAuctionExpired()` - уведомление об истечении аукциона

### 🔧 Изменения:
Заменено:
```javascript
const bot = require('./bot');
bot.sendMessage(...)
```

На:
```javascript
if (global.telegramBot) {
  global.telegramBot.sendMessage(...)
}
```

## Результат:
Теперь продавцы будут получать уведомления когда их предметы покупают в маркетплейсе:

```
✅ Ваш предмет куплен!

📦 Предмет: [название]
👤 Покупатель: [имя]
💰 Вы получили: [сумма] (комиссия 20%)

Деньги зачислены на ваш счет!
```
