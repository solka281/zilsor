# Функция выдачи расы админом

## Дата: 2026-04-22

## Описание

Добавлена возможность для админа выдавать любую расу любому игроку через админ панель.

## Изменения

### 1. Добавлена кнопка в админ панель
- В главном меню админ панели добавлена кнопка "🧬 Выдать расу"
- Расположена между кнопками выдачи кристаллов и сброса кулдаунов

### 2. Процесс выдачи расы

#### Шаг 1: Ввод ID игрока
- Админ нажимает "🧬 Выдать расу"
- Система запрашивает ID игрока
- Проверяется существование игрока в базе данных
- Если игрок не найден - показывается ошибка

#### Шаг 2: Выбор расы
- Показывается список всех рас с пагинацией (5 рас на страницу)
- Расы отсортированы по редкости (DESC) и названию (ASC)
- Каждая раса отображается с иконкой редкости
- Навигация стрелками ◀️ ▶️ для перелистывания страниц

#### Шаг 3: Подтверждение
- При выборе расы она сразу выдается игроку
- Админ получает подтверждение с информацией о выданной расе
- Игрок получает уведомление с полной информацией о расе

### 3. Уведомления

#### Админу:
```
✅ Раса выдана!

👤 Игрок: username
🧬 Раса: 🟡 Феникс
```

#### Игроку:
```
🎁 Вам выдана раса!

🟡 Феникс
⭐ Легендарная

⚡ Сила: 300
❤️ Здоровье: 200
🗡️ Атака: 80
🛡️ Защита: 60

🎯 Возрождение: Воскрешение после смерти
```

### 4. Логирование

В канал логов отправляется сообщение:
```
👑 Админ выдал расу
👤 Админ: 123456 (@admin_username)
🎯 Получатель: 789012 (@player_username)
🧬 Раса: 🟡 Феникс (Легендарная)
```

## Технические детали

### Новые callback handlers:
- `admin_give_race` - начало процесса выдачи расы
- `admin_race_page_X` - пагинация списка рас
- `admin_select_race_X` - выбор конкретной расы для выдачи

### Новые состояния админа:
- `awaiting_user_id_for_race` - ожидание ввода ID игрока
- `awaiting_race_selection` - ожидание выбора расы (с пагинацией)

### Изменения в коде:

1. **bot.js** - case 'admin_back':
   - Добавлена кнопка "🧬 Выдать расу" в админ панель

2. **bot.js** - case 'admin_give_race':
   - Новый обработчик для начала процесса выдачи расы

3. **bot.js** - текстовые сообщения:
   - Добавлен обработчик `awaiting_user_id_for_race`
   - Проверка существования игрока
   - Загрузка списка рас
   - Отображение первой страницы с пагинацией

4. **bot.js** - default section:
   - Добавлен обработчик `admin_race_page_X` для пагинации
   - Добавлен обработчик `admin_select_race_X` для выбора расы
   - Выдача расы игроку через UPDATE players
   - Отправка уведомлений админу и игроку
   - Логирование действия

## Использование

### Для админа:

1. Открыть админ панель (команда /admin или через меню)
2. Нажать "🧬 Выдать расу"
3. Ввести ID игрока (например: 123456789)
4. Выбрать расу из списка (листать стрелками если нужно)
5. Раса автоматически выдается игроку

### Примечания:

- Раса заменяет текущую расу игрока (если она была)
- Игрок сразу получает все характеристики новой расы
- Действие логируется в канал логов
- Можно выдать любую расу, включая легендарные и секретные

## Безопасность

- Все действия доступны только админу (проверка `config.ADMIN_USERNAME`)
- Проверяется существование игрока перед выдачей
- Проверяется существование расы перед выдачей
- Все действия логируются

## Пример использования

**Сценарий:** Админ хочет выдать игроку легендарную расу Феникс

1. Админ: Открывает админ панель
2. Админ: Нажимает "🧬 Выдать расу"
3. Бот: "Введите ID игрока:"
4. Админ: Вводит "123456789"
5. Бот: Показывает список рас с пагинацией
6. Админ: Листает до Феникса и нажимает "🟡 Феникс"
7. Бот админу: "✅ Раса выдана! 👤 Игрок: username 🧬 Раса: 🟡 Феникс"
8. Бот игроку: "🎁 Вам выдана раса! 🟡 Феникс ⭐ Легендарная ..."
9. Лог: "👑 Админ выдал расу ..."
