# Исправление системы прокачки расы

## Дата: 2026-04-22

## Проблема

Система прокачки расы не работала корректно:
- При нажатии на кнопку "⬆️ Прокачать расу" ничего не происходило
- Не было обработки ошибок
- Не было логирования для отладки
- После прокачки возвращалось неправильное меню

## Исправления

### 1. Улучшена обработка ошибок в `upgrade_race`

**Было:**
```javascript
case 'upgrade_race':
  getOrCreatePlayer(userId, query.from.username, (err, player) => {
    if (!player.race_id) {
      return bot.sendMessage(userId, '❌ Сначала получите расу!', getMainMenu(false));
    }
    // ...
  });
```

**Стало:**
```javascript
case 'upgrade_race':
  getOrCreatePlayer(userId, query.from.username, (err, player) => {
    if (err) {
      console.error('Ошибка получения игрока:', err);
      return bot.sendMessage(userId, '❌ Ошибка получения данных игрока');
    }
    
    if (!player.race_id) {
      return safeEditMessageText(chatId, messageId, 
        '❌ Сначала получите расу!', 
        getUpgradeMenu()
      );
    }
    
    console.log(`[UPGRADE_RACE] Игрок ${userId} пытается прокачать расу. Уровень: ${player.level}, Стоимость: ${cost}, Золото: ${player.gold}`);
    // ...
  });
```

**Изменения:**
- ✅ Добавлена проверка ошибки `err` при получении игрока
- ✅ Добавлено логирование для отладки
- ✅ Используется `safeEditMessageText` вместо `bot.sendMessage`
- ✅ Возвращается меню улучшений вместо главного меню

### 2. Улучшена обработка подтверждения в `confirm_upgrade_race`

**Было:**
```javascript
case 'confirm_upgrade_race':
  getOrCreatePlayer(userId, query.from.username, (err, player) => {
    if (!player.race_id) {
      return bot.sendMessage(userId, '❌ Сначала получите расу!', getMainMenu(false));
    }
    
    const cost = 100 * (player.level + 1);
    
    if (player.gold < cost) {
      return bot.sendMessage(userId, `❌ Недостаточно золота! Нужно: ${cost}💰`, getMainMenu());
    }
    
    db.run(`UPDATE players SET gold = gold - ?, power = power + 20 WHERE user_id = ?`,
      [cost, userId], () => {
        bot.sendMessage(userId, 
          `✅ Раса прокачана!\n\n` +
          `⚡ +20 к силе\n` +
          `💰 -${cost} золота`,
          getMainMenu()
        );
        // ...
      });
  });
```

**Стало:**
```javascript
case 'confirm_upgrade_race':
  getOrCreatePlayer(userId, query.from.username, (err, player) => {
    if (err) {
      console.error('Ошибка получения игрока:', err);
      return bot.sendMessage(userId, '❌ Ошибка получения данных игрока');
    }
    
    if (!player.race_id) {
      return safeEditMessageText(chatId, messageId, 
        '❌ Сначала получите расу!', 
        getUpgradeMenu()
      );
    }
    
    const cost = 100 * (player.level + 1);
    
    console.log(`[CONFIRM_UPGRADE_RACE] Игрок ${userId} подтверждает прокачку. Золото: ${player.gold}, Стоимость: ${cost}`);
    
    if (player.gold < cost) {
      return safeEditMessageText(chatId, messageId, 
        `❌ Недостаточно золота!\n\n` +
        `💰 Нужно: ${cost} золота\n` +
        `💵 У вас: ${player.gold} золота`,
        getUpgradeMenu()
      );
    }
    
    db.run(`UPDATE players SET gold = gold - ?, power = power + 20 WHERE user_id = ?`,
      [cost, userId], (updateErr) => {
        if (updateErr) {
          console.error('Ошибка обновления игрока:', updateErr);
          return bot.sendMessage(userId, '❌ Ошибка при прокачке расы');
        }
        
        console.log(`[CONFIRM_UPGRADE_RACE] Раса успешно прокачана для игрока ${userId}`);
        
        safeEditMessageText(chatId, messageId,
          `✅ *Раса прокачана!*\n\n` +
          `⚡ +20 к силе\n` +
          `💰 -${cost} золота\n\n` +
          `Новая сила: ${player.power + 20}`,
          {
            parse_mode: 'Markdown',
            ...getUpgradeMenu()
          }
        );
        
        // Улучшенное логирование
        sendLogMessage(
          `⬆️ Раса прокачана\n` +
          `👤 Игрок: ${userId} (@${query.from.username})\n` +
          `⚡ +20 к силе (${player.power} → ${player.power + 20})\n` +
          `💰 Потрачено: ${cost} золота`,
          'SUCCESS'
        );
      });
  });
```

**Изменения:**
- ✅ Добавлена проверка ошибки `err` при получении игрока
- ✅ Добавлена проверка ошибки `updateErr` при обновлении базы данных
- ✅ Добавлено логирование на каждом этапе
- ✅ Используется `safeEditMessageText` вместо `bot.sendMessage`
- ✅ Возвращается меню улучшений вместо главного меню
- ✅ Показывается новая сила после прокачки
- ✅ Улучшено сообщение об ошибке недостаточности золота
- ✅ Улучшено логирование с указанием username и изменения силы

## Возможные причины проблем

Если прокачка все еще не работает, проверьте:

1. **База данных:**
   - Есть ли у игрока раса (`race_id` не NULL)
   - Есть ли у игрока достаточно золота
   - Правильно ли работает UPDATE запрос

2. **Callback handlers:**
   - Правильно ли зарегистрированы обработчики `upgrade_race` и `confirm_upgrade_race`
   - Нет ли конфликтов с другими обработчиками

3. **Логи:**
   - Проверьте консоль на наличие ошибок
   - Проверьте, выводятся ли логи `[UPGRADE_RACE]` и `[CONFIRM_UPGRADE_RACE]`

4. **Меню:**
   - Правильно ли отображается кнопка "⬆️ Прокачать расу"
   - Правильно ли работает `getUpgradeMenu()`
