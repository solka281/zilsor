# Wiki - Исправление навигации

## ❌ Проблемы:
1. Кнопка "Главная" не работает - показывает предыдущую страницу
2. Навигация работает странно - кнопки показывают не ту страницу
3. Нельзя вернуться на стартовую страницу
4. "Загрузка данных..." показывается бесконечно

## ✅ Исправления:

### 1. Исправлена функция `showSection()`
**Проблема:** Удалялся `welcomeSection` из DOM, из-за чего нельзя было вернуться на главную

**Решение:**
- Убрал код который скрывал welcomeSection
- Добавил console.log для отладки
- Добавил обработку секции 'systems'
- Добавил default case для возврата на главную

### 2. Исправлена функция `showHomePage()`
**Проблема:** Использовался `appendChild` который ПЕРЕМЕЩАЛ элемент, а не копировал

**Решение:**
- Теперь ВСЕГДА создается новая главная страница через `innerHTML`
- Переустанавливаются обработчики событий для быстрых ссылок
- Главная страница всегда доступна

### 3. Добавлена кнопка "Главная" в навигацию
**Статус:** Уже была в index.html
```html
<div class="nav-section">
    <h3><i class="fas fa-home"></i> Главная</h3>
    <ul class="nav-list">
        <li><a href="#home" data-section="home">🏠 Главная страница</a></li>
    </ul>
</div>
```

## 🔍 Отладка "Загрузка данных..."

Проблема может быть в:
1. Данные не загружаются из файлов
2. Проверка `typeof RACE_DATA !== 'undefined'` не работает
3. Порядок загрузки скриптов

**Порядок загрузки (правильный):**
```html
<script src="race_data.js"></script>      <!-- 1. Данные рас -->
<script src="item_data.js"></script>      <!-- 2. Данные предметов -->
<script src="locations_data.js"></script> <!-- 3. Данные локаций -->
<script src="app.js"></script>            <!-- 4. Логика приложения -->
```

2. Проверить есть ли ошибки загрузки файлов
3. Проверить что `RACE_DATA`, `ITEM_DATA`, `LOCATIONS_DATA` определены:
   ```javascript
   console.log('RACE_DATA:', typeof RACE_DATA);
   console.log('ITEM_DATA:', typeof ITEM_DATA);
   console.log('LOCATIONS_DATA:', typeof LOCATIONS_DATA);
   ```

4. Проверить что файлы находятся в правильных путях:
