# Улучшенная система логирования

## Что изменено:

Добавлена функция `logWithPlayer()` которая автоматически получает и отображает ник игрока вместе с ID в логах консоли.

### Новая функция:
```javascript
function logWithPlayer(message, userId, callback) {
  if (!userId) {
    console.log(message);
    if (callback) callback();
    return;
  }
  
  db.get(`SELECT username, display_name FROM players WHERE user_id = ?`, [userId], (err, player) => {
    if (err || !player) {
      console.log(`${message} [ID: ${userId}]`);
    } else {
      const displayName = player.display_name || player.username || `User${userId}`;
      console.log(`${message} [${displayName} (${userId})]`);
    }
    if (callback) callback();
  });
}
```

## Обновленные места логирования:

1. **Окончание боя**: `handleBattleEnd`
2. **Выбор боссов**: `showRaidBossSelection`
3. **Callback обработчики**: основной обработчик callback_query
4. **Боевые действия**: атака, защита, спец. атака
5. **Поиск дуэлей**: все этапы matchmaking
6. **Квесты**: открытие меню, получение списка
7. **Прокачка расы**: попытка и подтверждение
8. **Текстовые сообщения**: обработка админских команд

## Преимущества:

✅ **Читаемость**: Сразу видно кто выполняет действие
✅ **Отладка**: Легче отслеживать действия конкретных игроков  
✅ **Мониторинг**: Удобно следить за активностью игроков
✅ **Поддержка**: Быстрее помочь игроку при проблемах

## Формат логов:

- **С ником**: `[Mnxanj (6587524595)]`
- **Без ника**: `[ID: 6587524595]` (если игрок не найден)
- **Без ID**: обычный лог (если userId не передан)
