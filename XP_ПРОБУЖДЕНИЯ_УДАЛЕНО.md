# XP Пробуждения - Удалено

## Что изменено:

### ❌ Удалено из дуэлей:
- XP пробуждения больше не начисляется за победы/поражения в дуэлях
- Убрано отображение "+100/+20 XP пробуждения" из результатов дуэли

### ❌ Удалены все бонусы пробуждения:
- Бонусы к золоту и опыту от уровня пробуждения отключены
- Убраны из боевой системы (лес)
- Убраны из работы
- Убраны из квестов
- Убраны из дуэлей (duels.js)

### ❌ Удалено из интерфейса:
- XP пробуждения убрано из профиля игрока
- XP пробуждения убрано из статистики
- XP пробуждения убрано из поиска сокровищ

### ✅ Система пробуждения упрощена:
- Пробуждение теперь требует только золото (500💰)
- Убрана проверка XP пробуждения
- Убрано требование фарма XP
- Пробуждение дает только +50 к силе

### 🔧 Технические изменения:
- Функция `applyAwakeningBonus()` удалена
- SQL запросы обновлены (убрано `awakening_xp`)
- Убраны все упоминания бонусов пробуждения из сообщений

## Результат:
Теперь пробуждение расы - это простая покупка за золото без необходимости фарма XP пробуждения.
