# Система характеристик

## Замена древа навыков

Система "Древо навыков" заменена на "Характеристики" - более простую и понятную систему прокачки.

## Характеристики

### ❤️ Здоровье (HP)
- **Базовая стоимость**: 80 золота
- **Рост стоимости**: x1.18 за каждый уровень
- **Прирост за уровень**: 10 + (уровень × 2)
- **Формула бонуса**: `10 + (level * 2)`

### ⚔️ Урон (Attack)
- **Базовая стоимость**: 100 золота
- **Рост стоимости**: x1.20 за каждый уровень
- **Прирост за уровень**: 6 + (уровень × 1)
- **Формула бонуса**: `6 + (level * 1)`

### 🛡️ Защита (Defense)
- **Базовая стоимость**: 120 золота
- **Рост стоимости**: x1.22 за каждый уровень
- **Прирост за уровень**: 2 + (уровень × 0.5)
- **Формула бонуса**: `2 + (level * 0.5)`

## Формулы расчета

### Стоимость улучшения:
```javascript
cost = Math.floor(base_cost * Math.pow(cost_multiplier, current_level))
```

### Бонус от уровня:
```javascript
bonus = Math.floor(base_bonus + (level * level_multiplier))
```

## Техническая реализация

### База данных:
```sql
CREATE TABLE player_characteristics (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  player_id INTEGER NOT NULL,
  characteristic_type TEXT NOT NULL,
  level INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(player_id, characteristic_type)
);
```

### Интеграция с боевой системой:
- Бонусы автоматически учитываются в `calculatePlayerPower()`
- Влияют на все виды боев: PvE, PvP, рейды
- Отображаются в профиле игрока

## Преимущества новой системы

1. **Простота**: Понятные характеристики вместо сложного древа
2. **Гибкость**: Игрок сам выбирает что прокачивать
3. **Прогрессия**: Экспоненциальный рост стоимости
4. **Баланс**: Разные мультипликаторы для разных характеристик
5. **Интеграция**: Полная совместимость с существующими системами

## Миграция

- Старая система навыков остается в коде для совместимости
- Новые игроки используют только систему характеристик
- Существующие навыки продолжают работать
- Постепенный переход на новую систему
