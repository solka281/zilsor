# 🤖 Система автопродажи предметов

## 📋 Описание:
Система автоматической продажи предметов по редкости. Игроки могут настроить автопродажу для определенных типов редкости, и предметы будут автоматически продаваться при получении.

## ⚙️ Функционал:

### Настройки по редкости:
- ⚪ **Обычные** (COMMON) - 10💰
- 🔵 **Редкие** (RARE) - 25💰  
- 🟣 **Эпические** (EPIC) - 75💰
- 🟠 **Мифические** (MYTHIC) - 200💰
- 🟡 **Легендарные** (LEGENDARY) - 500💰

### Возможности:
1. **Включение/отключение** автопродажи для каждой редкости
2. **Мгновенная продажа** всех предметов по настройкам
3. **Статистика** - показывает количество предметов и потенциальное золото
4. **Автоматическая продажа** при получении новых предметов

## 🎮 Интерфейс:

### Доступ к меню:
`🎒 Инвентарь` → `🤖 Автопродажа`

### Кнопки меню:
- `✅/❌ [Редкость]` - переключить автопродажу для редкости
- `💰 Продать сейчас` - продать все предметы по текущим настройкам
- `📊 Статистика` - подробная информация о предметах и ценах

## 🔧 Техническая реализация:

### База данных:
```sql
CREATE TABLE auto_sell_settings (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  player_id INTEGER UNIQUE,
  common_enabled INTEGER DEFAULT 0,
  rare_enabled INTEGER DEFAULT 0,
  epic_enabled INTEGER DEFAULT 0,
  mythic_enabled INTEGER DEFAULT 0,
  legendary_enabled INTEGER DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

### Основные функции:
- `getAutoSellSettings()` - получить настройки игрока
- `updateAutoSellSettings()` - обновить настройку для редкости
- `checkAutoSell()` - проверить и продать предмет при получении
- `autoSellInventory()` - продать все предметы по настройкам
- `getAutoSellStats()` - получить статистику

### Интеграция:
Система интегрирована в `items.addItemToInventory()` - каждый новый предмет автоматически проверяется на автопродажу.

## 💡 Примеры использования:

### Сценарий 1: Настройка автопродажи
1. Игрок заходит в `🤖 Автопродажа`
2. Включает автопродажу для обычных предметов
3. Теперь все обычные предметы автоматически продаются за 10💰

### Сценарий 2: Получение предмета
1. Игрок находит обычный меч при поиске сокровищ
2. Система проверяет настройки автопродажи
3. Автопродажа включена для обычных предметов
4. Предмет автоматически продается за 10💰
5. Игрок видит: "🤖 Автопродажа: +10 золота"

### Сценарий 3: Массовая продажа
1. У игрока накопилось много предметов
2. Он настраивает автопродажу для нужных редкостей
3. Нажимает "💰 Продать сейчас"
4. Все предметы выбранных редкостей продаются мгновенно

## 📊 Статистика:

### Информация в меню:
- Текущие настройки для каждой редкости
- Количество предметов каждой редкости в инвентаре
- Потенциальное золото от продажи
- Цены продажи для каждой редкости

### Пример отображения:
```
🤖 Автопродажа предметов

⚙️ Настройки по редкости:
⚪ Обычные: ✅ (15 шт, 10💰)
🔵 Редкие: ❌ (3 шт, 25💰)
🟣 Эпические: ❌ (1 шт, 75💰)
🟠 Мифические: ❌ (0 шт, 200💰)
🟡 Легендарные: ❌ (0 шт, 500💰)

💰 Потенциальное золото: 150💰
```

## 🎯 Преимущества:

### Для игроков:
- **Автоматизация** - не нужно вручную продавать ненужные предметы
- **Экономия времени** - инвентарь не засоряется
- **Гибкость** - можно настроить под свои потребности
- **Прозрачность** - видно что и за сколько продается

### Для игры:
- **Улучшение UX** - меньше рутинных действий
- **Баланс экономики** - стабильные цены на предметы
- **Активность** - игроки больше времени тратят на интересный контент

## 🔄 Интеграция с другими системами:

### Поиск сокровищ:
При нахождении предмета автоматически проверяется автопродажа

### Награды за квесты:
Предметы из наград также проходят проверку автопродажи

### Рейды и битвы:
Все получаемые предметы обрабатываются системой

## ⚠️ Важные моменты:

1. **Экипированные предметы** не продаются автоматически
2. **Настройки индивидуальны** для каждого игрока
3. **Цены фиксированы** и не зависят от характеристик предмета
4. **Операция необратима** - проданные предметы нельзя вернуть

## 🚀 Результат:
Система автопродажи значительно улучшает пользовательский опыт, автоматизируя рутинные действия и позволяя игрокам сосредоточиться на интересном контенте!
