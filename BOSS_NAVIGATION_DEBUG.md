# Отладка навигации по боссам

## Добавлена отладочная информация

В функцию `showRaidBossSelection` добавлено подробное логирование:

### Что логируется:
1. **Вызов функции**: `showRaidBossSelection вызвана: currentIndex=X, userId=Y`
2. **Активный рейд**: Есть ли активный рейд
3. **Список боссов**: Количество найденных боссов и их данные
4. **Навигация**: Проверка условий для кнопок ◀️ ▶️
5. **Клавиатура**: Количество рядов кнопок и их содержимое

### Как тестировать:

1. **Откройте меню рейдов**:
   - Главное меню → "🐉 Рейды"
   - Нажмите "👹 Выбрать босса"

2. **Проверьте логи**:
   В консоли должно появиться:
   ```
   🎯 showRaidBossSelection вызвана: currentIndex=0, userId=123456
   🔍 Активного рейда нет, получаем список боссов...
   🔍 Получение списка боссов...
   📋 Найдено боссов в БД: 2
     1. Повелитель ветра (Уровень 1, ID: 1)
     2. Владыка тьмы (Уровень 2, ID: 2)
   🐉 Найдено боссов: 2
     1. Повелитель ветра (Уровень 1)
     2. Владыка тьмы (Уровень 2)
   📍 Текущий индекс: 0, всего боссов: 2
   ✅ Рейд готов, показываем выбор босса
   🎯 Показываем босса: индекс 0, босс: Повелитель ветра
   🔍 Проверяем навигацию: currentIndex=0, bosses.length=2
   ➡️ Добавляем кнопку "вперед"
   ✅ Добавляем 1 навигационных кнопок
   🎮 Отправляем клавиатуру с 3 рядами кнопок
     Ряд 1: ▶️
     Ряд 2: ⚔️ Создать рейд с этим боссом
     Ряд 3: 🔙 Назад
   ```

3. **Ожидаемое поведение**:
   - На первом боссе (Повелитель ветра): только кнопка ▶️
   - На втором боссе (Владыка тьмы): только кнопка ◀️
   - Если боссов больше 2: обе кнопки ◀️ ▶️

### Возможные проблемы:

1. **Боссов меньше 2**: 
   - Лог покажет `📋 Найдено боссов в БД: 1`
   - Навигационные кнопки не добавятся

2. **currentIndex неправильный**:
   - Лог покажет неправильный индекс
   - Проверить вызов `showRaidBossSelection(chatId, messageId, 0, userId)`

3. **Условие навигации не срабатывает**:
   - Лог покажет `❌ Навигационные кнопки не добавлены`
   - Проверить логику `currentIndex > 0` и `currentIndex < bosses.length - 1`

### Если проблема остается:

Отправьте логи из консоли, особенно строки:
- `📋 Найдено боссов в БД: X`
- `📍 Текущий индекс: X, всего боссов: Y`
- `🔍 Проверяем навигацию: currentIndex=X, bosses.length=Y`
- `🎮 Отправляем клавиатуру с X рядами кнопок`
