# ⚡ Как работает система скорости в Покемонах

## 🎯 ОСНОВНОЙ ПРИНЦИП:

**Скорость определяет порядок ходов в бою!**  
Покемон с более высокой скоростью атакует первым.

## 📊 МЕХАНИКА СКОРОСТИ:

### 1. **Сравнение скорости:**
- В начале каждого хода сравниваются характеристики Speed всех покемонов
- Покемон с **самой высокой скоростью** ходит первым
- Если скорость одинаковая → **случайный выбор** (50/50)

### 2. **Пример:**
```
Пикачу (Speed: 90) VS Чаризард (Speed: 100)
→ Чаризард атакует первым!

Пикачу (Speed: 90) VS Блейстойз (Speed: 78)  
→ Пикачу атакует первым!

Пикачу (Speed: 90) VS Пикачу (Speed: 90)
→ Случайный выбор кто первый
```

## ⚡ ПРИОРИТЕТ АТАК:

### Система приоритета важнее скорости!

**Уровни приоритета:**
- **+3:** Helping Hand
- **+2:** Protect, Detect  
- **+1:** Quick Attack, Mach Punch, Bullet Punch
- **0:** Обычные атаки (здесь работает скорость)
- **-1:** Vital Throw
- **-6:** Whirlwind, Roar

### Правило:
1. **Сначала** сравнивается приоритет атаки
2. **Потом** (при равном приоритете) сравнивается скорость

### Пример:
```
Медленный покемон использует Quick Attack (+1 приоритет)
VS
Быстрый покемон использует Tackle (0 приоритет)

→ Медленный покемон атакует первым!
```

## 🔄 ДИНАМИЧЕСКАЯ СКОРОСТЬ (Gen 8+):

### В новых играх (Sword/Shield+):
- Изменения скорости применяются **немедленно** в том же ходу
- Раньше изменения действовали только со следующего хода

### Пример:
```
Ход 1: Гримснарл использует Thunder Wave на Драгапульта
→ Драгапульт парализован (скорость уменьшена в 2 раза)
→ Дуралудон теперь быстрее и атакует до Драгапульта!

В старых играх: Драгапульт все равно атаковал бы первым в этом ходу
```

## 🎮 СПЕЦИАЛЬНЫЕ ЭФФЕКТЫ:

### 1. **Trick Room:**
- Медленные покемоны ходят первыми
- Быстрые покемоны ходят последними
- Приоритет все еще работает

### 2. **Предметы:**
- **Choice Scarf:** +50% к скорости
- **Quick Claw:** 20% шанс атаковать первым
- **Lagging Tail:** Всегда атакует последним

### 3. **Способности:**
- **Swift Swim:** x2 скорость под дождем
- **Chlorophyll:** x2 скорость на солнце  
- **Speed Boost:** +1 к скорости каждый ход

### 4. **Статусы:**
- **Паралич:** -50% к скорости
- **Tailwind:** x2 скорость команды на 4 хода

## 🧮 РАСЧЕТ СКОРОСТИ:

### Формула (упрощенно):
```
Итоговая скорость = (Базовая скорость + IV + EV бонусы) × Модификаторы

Модификаторы:
- Природа: ±10%
- Предметы: Choice Scarf (+50%)
- Способности: Swift Swim (x2)
- Статусы: Паралич (-50%)
```

## 🎯 СТРАТЕГИЧЕСКОЕ ЗНАЧЕНИЕ:

### Почему скорость важна:
1. **Контроль боя:** Атака первым = преимущество
2. **Нокаут до атаки:** Убить врага до его хода
3. **Защитные маневры:** Heal/Protect до получения урона
4. **Momentum:** Контроль темпа боя

### Speed Tiers (уровни скорости):
- **Очень быстрые:** 130+ (Драгапульт, Нинджаск)
- **Быстрые:** 100-129 (Чаризард, Генгар)
- **Средние:** 70-99 (Пикачу, Люкарио)  
- **Медленные:** 50-69 (Блейстойз, Венузавр)
- **Очень медленные:** <50 (Слакинг, Шакл)

## 🔄 СРАВНЕНИЕ С НАШИМ БОТОМ:

### В Покемонах:
- ✅ Скорость определяет порядок ходов
- ✅ Быстрый покемон атакует первым
- ✅ Есть система приоритета атак
- ✅ Динамические изменения скорости

### В нашем боте:
- ❌ Скорость НЕ влияет на порядок ходов
- ❌ Порядок фиксированный (Игрок → Враг)
- ✅ "Скорость" дает бонус к уклонению
- ❌ Нет системы приоритета

## 💡 ВЫВОД:

**В покемонах скорость работает как настоящая скорость** - определяет кто ходит первым. Это одна из самых важных характеристик в игре!

**В нашем боте "скорость" - это просто бонус к уклонению**, что не соответствует ожиданиям игроков, знакомых с покемонами.
