# Исправления: Характеристики и Выбор Боссов

## Проблема 1: Не работают кнопки апгрейда характеристик

### Причина:
Обработчики `upgrade_char_` находились после `break;` statement и не внутри switch case, поэтому никогда не выполнялись.

### Решение:
Перенесены обработчики в `default:` case внутри switch statement:

```javascript
default:
  // Обработчики улучшения характеристик
  if (data.startsWith('upgrade_char_')) {
    const charType = data.replace('upgrade_char_', '');
    
    characteristics.upgradeCharacteristic(userId, charType, (err, result) => {
      // ... логика улучшения
    });
    return;
  }
```

### Результат:
- ✅ Кнопки улучшения характеристик теперь работают
- ✅ Показывается уведомление об успешном улучшении
- ✅ Меню автоматически обновляется после улучшения

## Проблема 2: В выборе боссов показывается только Владыка тьмы

### Причина:
1. Повелитель ветра имел уровень 9, а Владыка тьмы - уровень 2
2. При сортировке по уровню Владыка тьмы шел первым
3. Возможно, показывался только первый босс из списка

### Решение:
1. **Исправлена сортировка боссов** по уровню в `showRaidBossSelection()`:
```javascript
// Сортируем боссов по уровню (сначала 1 уровень, потом 2 и т.д.)
bosses.sort((a, b) => a.level - b.level);
```

2. **Изменен уровень Повелителя ветра** с 9 на 1:
```javascript
// В raids.js
['Повелитель ветра', 1, 75000, 'raid_wind_lord.jpg', ...]
```

### Результат:
- ✅ Боссы сортируются по уровню: сначала уровень 1, потом 2
- ✅ Повелитель ветра (уровень 1) показывается первым
- ✅ Владыка тьмы (уровень 2) показывается вторым
- ✅ Навигация стрелочками работает между боссами
