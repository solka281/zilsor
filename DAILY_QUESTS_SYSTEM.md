# Система ежедневных квестов

## Описание
Добавлена полноценная система ежедневных квестов с базой из 100 квестов. Каждому игроку ежедневно выдается 3 случайных квеста.

## Возможности

### 1. База квестов (100 штук)
Квесты разделены по категориям:

#### Дуэли (20 квестов)
- Победы в дуэлях (1-6 побед)
- Участие в дуэлях (2-7 дуэлей)
- Серии побед (3-4 подряд)

#### Темный лес (20 квестов)
- Победы над врагами (1-10 побед)
- Различные уровни сложности
- Прогрессивные награды

#### Поиск предметов (15 квестов)
- Найти предметы (1-10 штук)
- Награды за удачу

#### Прокачка (15 квестов)
- Прокачать расу (1-6 раз)
- Развитие персонажа

#### Работа (10 квестов)
- Поработать (1-5 раз)
- Трудовые достижения

#### Золото (10 квестов)
- Накопить золото (500-5000)
- Финансовые цели

#### Комбинированные (10 квестов)
- Выполнить несколько типов заданий
- Универсальные квесты

### 2. Ежедневная выдача
- Каждый день игрок получает 3 случайных квеста
- Квесты выдаются автоматически при открытии меню квестов
- Квесты обновляются в 00:00 по серверному времени

### 3. Прогресс квестов
- Автоматическое отслеживание прогресса
- Прогресс-бар для каждого квеста
- Уведомления о выполнении

### 4. Награды
- Золото: 40-300 за квест
- Опыт: 25-200 за квест
- Автоматическая выдача при выполнении

## Структура базы данных

### Таблица `daily_quests_pool`
```sql
CREATE TABLE daily_quests_pool (
  id INTEGER PRIMARY KEY,
  name TEXT UNIQUE,
  description TEXT,
  type TEXT,
  requirement INTEGER,
  reward_gold INTEGER,
  reward_exp INTEGER
)
```

### Таблица `player_daily_quests`
```sql
CREATE TABLE player_daily_quests (
  id INTEGER PRIMARY KEY,
  player_id INTEGER,
  quest_id INTEGER,
  quest_name TEXT,
  quest_description TEXT,
  quest_type TEXT,
  requirement INTEGER,
  reward_gold INTEGER,
  reward_exp INTEGER,
  progress INTEGER DEFAULT 0,
  completed INTEGER DEFAULT 0,
  assigned_at DATETIME,
  completed_at DATETIME
)
```

## Типы квестов

| Тип | Описание | Пример |
|-----|----------|--------|
| `duel_win` | Победы в дуэлях | Победите в 3 дуэлях |
| `duel_participate` | Участие в дуэлях | Проведите 5 дуэлей |
| `duel_streak` | Серия побед | Победите в 4 дуэлях подряд |
| `forest_win` | Победы в лесу | Победите 5 врагов в темном лесу |
| `loot` | Поиск предметов | Найдите 3 предмета |
| `upgrade` | Прокачка расы | Прокачайте расу 2 раза |
| `work` | Работа | Поработайте 3 раза |
| `gold_total` | Накопление золота | Накопите 1000 золота |
| `combo_*` | Комбинированные | Различные комбинации |

## API модуля daily_quests.js

### `initializeDailyQuests()`
Инициализирует базу из 100 квестов

### `assignDailyQuests(playerId, callback)`
Выдает 3 случайных квеста игроку
- Проверяет есть ли уже квесты на сегодня
- Выбирает 3 случайных квеста из пула
- Сохраняет в player_daily_quests

### `getPlayerDailyQuests(playerId, callback)`
Получить активные квесты игрока на сегодня

### `updateQuestProgress(playerId, questType, amount, callback)`
Обновить прогресс квеста
- Находит квесты нужного типа
- Увеличивает прогресс
- Проверяет выполнение
- Выдает награды

## Интеграция

### bot.js
**Обработчик квестов:**
- Автоматическая выдача при открытии меню
- Отображение прогресса с прогресс-баром
- Показ наград

**Обновление прогресса:**
- После победы в дуэли → `duel_win`, `duel_participate`
- После победы в лесу → `forest_win`
- После поиска предмета → `loot`
- После прокачки → `upgrade`
- После работы → `work`

## Процесс использования

### 1. Получение квестов
```
Игрок: Битвы → Квесты
Система: Выдает 3 случайных квеста
Результат: Квесты отображаются с прогрессом 0/X
```

### 2. Выполнение
```
Игрок: Побеждает в дуэли
Система: Обновляет прогресс квеста "Победите в 3 дуэлях" (1/3)
Игрок: Побеждает еще в 2 дуэлях
Система: Квест выполнен! (3/3)
Результат: Выдаются награды
```

### 3. Уведомление о выполнении
```
✅ Квест выполнен!

📋 Боец дня
🎁 +100💰 +60✨
```

### 4. Обновление на следующий день
```
00:00 - Старые квесты удаляются
Игрок: Открывает меню квестов
Система: Выдает 3 новых случайных квеста
```

## Особенности

✅ **Автоматическая выдача:** Квесты выдаются при открытии меню
✅ **Случайность:** Каждый день новые квесты
✅ **Прогресс:** Автоматическое отслеживание
✅ **Уведомления:** Сообщения о выполнении
✅ **Разнообразие:** 100 различных квестов
✅ **Баланс:** Награды соответствуют сложности

## Балансировка наград

### По сложности
- Легкие (1-2 действия): 40-80 золота, 25-50 опыта
- Средние (3-5 действий): 90-150 золота, 55-100 опыта
- Сложные (6-10 действий): 180-300 золота, 120-200 опыта

### По типу
- Дуэли: Высокие награды (боевая активность)
- Лес: Средние награды (PvE контент)
- Поиск: Средние награды (удача)
- Прокачка: Высокие награды (прогресс)
- Работа: Низкие награды (пассивный доход)
- Комбо: Высокие награды (сложность)
