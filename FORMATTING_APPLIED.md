## Правила форматирования

- **< 1,000**: обычный формат (500)
- **1,000 - 99,999**: с пробелами (15 000)
- **100,000+**: с сокращениями (150K, 1.5M, 2.5B)

## Что еще можно отформатировать

Если нужно добавить форматирование в других местах, используйте:

```javascript
smartFormat(число)
```

Например:
```javascript
`💰 Золото: ${smartFormat(player.gold)}`
`❤️ HP: ${smartFormat(stats.hp)}`
`⚔️ Урон: ${smartFormat(damage)}`
```

## Откат изменений

Если нужно вернуть старое поведение, замените:
```javascript
smartFormat(число)
```

На:
```javascript
число.toLocaleString()  // или просто число
```
