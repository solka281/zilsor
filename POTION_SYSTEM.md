# Система зелий

## Описание
Добавлена полноценная система зелий с магазином, инвентарем и использованием в бою.

## Возможности

### 1. Магазин зелий
- Доступ: Главное меню → 🛒 Магазин → 🧪 Зелья
- Покупка зелий за золото
- Выбор количества: 1, 5, 10, 20 штук

### 2. Доступные зелья

| Зелье | Эффект | Цена |
|-------|--------|------|
| 🧪 Малое зелье жизни | Восстанавливает 30% HP | 100 золота |
| 🧪 Большое зелье жизни | Восстанавливает 60% HP | 250 золота |

### 3. Инвентарь зелий
- Доступ: Инвентарь → 🧪 Зелья
- Просмотр всех зелий
- Экипировка/снятие зелья
- Покупка дополнительных зелий

### 4. Слот экипировки
- Новый слот "🧪 Зелье" в экипировке
- Можно экипировать только 1 зелье
- Показывает название и количество

### 5. Использование в бою
- Кнопка "🧪 [Название] (X шт)" появляется в бою
- Восстанавливает HP согласно эффекту зелья
- Тратится 1 зелье за использование
- Враг атакует после использования

## Структура базы данных

### Таблица `potions`
```sql
CREATE TABLE potions (
  id INTEGER PRIMARY KEY,
  name TEXT UNIQUE,
  description TEXT,
  type TEXT,
  heal_amount INTEGER,  -- процент восстановления HP
  price INTEGER
)
```

### Таблица `player_potions`
```sql
CREATE TABLE player_potions (
  id INTEGER PRIMARY KEY,
  player_id INTEGER,
  potion_id INTEGER,
  quantity INTEGER DEFAULT 0,
  UNIQUE(player_id, potion_id)
)
```

### Обновление таблицы `players`
```sql
ALTER TABLE players ADD COLUMN equipped_potion_id INTEGER DEFAULT NULL
```

## API модуля potions.js

### `initializePotions()`
Инициализирует зелья в базе данных

### `getAllPotions(callback)`
Получить все доступные зелья

### `getPotionById(potionId, callback)`
Получить зелье по ID

### `getPlayerPotions(playerId, callback)`
Получить инвентарь зелий игрока

### `buyPotion(playerId, potionId, quantity, callback)`
Купить зелье
- Проверяет наличие золота
- Списывает золото
- Добавляет зелье в инвентарь

### `equipPotion(playerId, potionId, callback)`
Экипировать зелье
- Проверяет наличие зелья
- Устанавливает equipped_potion_id

### `unequipPotion(playerId, callback)`
Снять зелье

### `usePotion(playerId, callback)`
Использовать зелье в бою
- Проверяет наличие экипированного зелья
- Уменьшает количество на 1
- Возвращает информацию о зелье
- Автоматически снимает если закончилось

### `getPotionQuantity(playerId, potionId, callback)`
Получить количество конкретного зелья

## Процесс использования

### Покупка
1. Магазин → Зелья
2. Выбрать зелье
3. Выбрать количество (1/5/10/20)
4. Подтвердить покупку
5. Золото списывается, зелья добавляются

### Экипировка
1. Инвентарь → Зелья
2. Выбрать зелье
3. Нажать "✅ Экипировать"
4. Зелье появится в слоте экипировки

### Использование в бою
1. Начать любой бой (лес/дуэль)
2. Увидеть кнопку "🧪 [Название] (X шт)"
3. Нажать кнопку
4. HP восстанавливается
5. Количество уменьшается на 1
6. Враг атакует

## Особенности

✅ **Расходуемость:** Зелья тратятся только при использовании
✅ **Выбор игрока:** Игрок сам решает когда использовать
✅ **Автоматическое снятие:** Если зелья закончились, автоматически снимается
✅ **Показ количества:** В бою видно сколько зелий осталось
✅ **Защита от ошибок:** Нельзя использовать если не экипировано или закончилось

## Примеры использования

### Пример 1: Покупка малого зелья
```
Игрок: Магазин → Зелья → Малое зелье жизни → 5 шт
Система: Списывает 500 золота (100 × 5)
Результат: +5 малых зелий в инвентаре
```

### Пример 2: Использование в бою
```
Игрок: Экипирует большое зелье (3 шт)
Игрок: Идет в темный лес
Игрок: HP 50/200 → Использует зелье
Система: Восстанавливает 120 HP (60% от 200)
Результат: HP 170/200, осталось 2 зелья
```

### Пример 3: Автоматическое снятие
```
Игрок: Экипировано малое зелье (1 шт)
Игрок: Использует последнее зелье в бою
Система: Восстанавливает HP
Система: Автоматически снимает зелье (quantity = 0)
Результат: Слот зелья пуст
```

## Балансировка

### Цены
- Малое зелье: 100 золота (30% HP)
- Большое зелье: 250 золота (60% HP)

### Эффективность
- Малое: ~3.33 золота за 1% HP
- Большое: ~4.17 золота за 1% HP
- Малое более эффективно, но восстанавливает меньше

### Рекомендации
- Малое зелье: для частого использования
- Большое зелье: для сложных боссов
