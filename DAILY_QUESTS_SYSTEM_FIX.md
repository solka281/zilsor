# Исправление системы ежедневных квестов

## Дата: 2026-04-22

## Проблемы

1. **Не видно прогресс квестов** - квесты не обновлялись в реальном времени
2. **Квесты не выполняются** - выполненные квесты не удалялись из списка
3. **Выполняется квест которого нет в меню** - старые квесты не удалялись
4. **Нет поддержки VIP** - все игроки получали 3 квеста вместо 5 для VIP

## Исправления

### 1. Добавлена поддержка VIP игроков

**Файл:** `daily_quests.js`

**Было:**
```javascript
// Выдать 3 случайных квеста игроку на день
function assignDailyQuests(playerId, callback) {
  // ...
  // Если уже есть 3 квеста - не выдаем новые
  if (currentQuests >= 3) {
    // ...
  }
  // Получаем 3 случайных квеста
  db.all(`SELECT * FROM daily_quests_pool ORDER BY RANDOM() LIMIT 3`, ...);
}
```

**Стало:**
```javascript
// Выдать случайные квесты игроку на день (3 для обычных, 5 для VIP)
function assignDailyQuests(playerId, callback) {
  // Получаем информацию об игроке для проверки VIP статуса
  db.get(`SELECT vip FROM players WHERE user_id = ?`, [playerId], (err, player) => {
    const isVip = player && player.vip === 1;
    const questCount = isVip ? 5 : 3;
    
    // Если уже есть нужное количество квестов - не выдаем новые
    if (currentQuests >= questCount) {
      // ...
    }
    
    // Получаем случайные квесты
    db.all(`SELECT * FROM daily_quests_pool ORDER BY RANDOM() LIMIT ?`, [questCount], ...);
  });
}
```

**Изменения:**
- ✅ Проверка VIP статуса игрока
- ✅ Динамическое количество квестов (3 или 5)
- ✅ Логирование VIP статуса

### 2. Выполненные квесты теперь удаляются из списка

**Файл:** `daily_quests.js`

**Было:**
```javascript
if (newProgress >= quest.requirement && quest.completed === 0) {
  // Квест выполнен!
  db.run(`UPDATE player_daily_quests SET progress = ?, completed = 1, completed_at = CURRENT_TIMESTAMP 
          WHERE id = ?`, [newProgress, quest.id], (err) => {
    // Выдаем награду
    db.run(`UPDATE players SET gold = gold + ?, exp = exp + ? WHERE user_id = ?`,
      [quest.reward_gold, quest.reward_exp, playerId], (err) => {
        // Квест остается в таблице с completed = 1
      });
  });
}
```

**Стало:**
```javascript
if (newProgress >= quest.requirement && quest.completed === 0) {
  // Квест выполнен!
  db.run(`UPDATE player_daily_quests SET progress = ?, completed = 1, completed_at = CURRENT_TIMESTAMP 
          WHERE id = ?`, [newProgress, quest.id], (err) => {
    // Выдаем награду
    db.run(`UPDATE players SET gold = gold + ?, exp = exp + ? WHERE user_id = ?`,
      [quest.reward_gold, quest.reward_exp, playerId], (err) => {
        // Удаляем выполненный квест из таблицы
        db.run(`DELETE FROM player_daily_quests WHERE id = ?`, [quest.id], (err) => {
          console.log(`🗑️ Выполненный квест "${quest.quest_name}" удален из списка`);
        });
      });
  });
}
```

**Изменения:**
- ✅ Выполненные квесты удаляются из таблицы
- ✅ Добавлено логирование удаления
- ✅ Квесты исчезают из списка после выполнения

### 4. Добавлено обновление квестов при прокачке расы

**Файл:** `bot.js`

**Было:**
```javascript
case 'confirm_upgrade_race':
  db.run(`UPDATE players SET gold = gold - ?, power = power + 20 WHERE user_id = ?`,
    [cost, userId], (updateErr) => {
      // Логируем прокачку расы
      sendLogMessage(...);
      // Квесты не обновлялись
    });
```

**Стало:**
```javascript
case 'confirm_upgrade_race':
  db.run(`UPDATE players SET gold = gold - ?, power = power + 20 WHERE user_id = ?`,
    [cost, userId], (updateErr) => {
      // Логируем прокачку расы
      sendLogMessage(...);
      
      // Обновляем прогресс квестов на прокачку
      dailyQuests.updateQuestProgress(userId, 'upgrade', 1, (err, completedQuests) => {
        if (completedQuests && completedQuests.length > 0) {
          completedQuests.forEach(quest => {
            bot.sendMessage(userId, 
              `✅ Квест выполнен!\n\n` +
              `📋 ${quest.quest_name}\n` +
              `🎁 +${quest.reward_gold}💰 +${quest.reward_exp}✨`
            );
            
            // Логируем выполнение квеста
            sendLogMessage(...);
          });
        }
      });
    });
```

**Изменения:**
- ✅ Квесты на прокачку расы теперь обновляются
- ✅ Игрок получает уведомление о выполнении квеста
- ✅ Выполнение квеста логируется

## Типы квестов

Система поддерживает следующие типы квестов:

1. **duel_win** - победы в дуэлях
2. **duel_participate** - участие в дуэлях
3. **duel_streak** - серия побед в дуэлях
4. **forest_win** - победы в темном лесу
5. **loot** - поиск предметов
6. **upgrade** - прокачка расы ✅ ИСПРАВЛЕНО
7. **work** - работа ✅ УЖЕ РАБОТАЛО
8. **gold_total** - накопление золота
9. **combo_*** - комбинированные квесты

## Обновление квестов

Квесты обновляются каждый день в **00:00 по МСК**.

### Логика обновления:
1. При открытии меню квестов проверяется дата
2. Если квесты старые (не сегодняшние) - они удаляются
3. Если квестов нет или меньше нужного - выдаются новые
4. Количество квестов зависит от VIP статуса (3 или 5)
