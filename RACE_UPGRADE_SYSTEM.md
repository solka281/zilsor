# Система прокачки расы

## Дата: 2026-04-22

## Описание

Реализована полная система прокачки расы, позволяющая игрокам улучшать свою расу за золото, получая бонусы к силе.

## Основные возможности

### 1. Прокачка расы (Race Upgrade)
- Доступна через меню "⬆️ Улучшения" → "⬆️ Прокачать расу"
- Требует наличие расы у игрока
- Стоимость зависит от уровня игрока: `100 * (уровень + 1)` золота
- Бонус: +20 к силе за каждую прокачку
- Нет ограничений на количество прокачек

### 2. Пробуждение расы (Race Awakening)
- Доступно через меню "⬆️ Улучшения" → "🌟 Пробудить расу"
- Требует наличие расы у игрока
- Максимальный уровень пробуждения: 10
- Требования для пробуждения:
  - XP пробуждения: 1000 (настраивается в config.AWAKENING_XP_REQUIRED)
  - Золото: 500 (настраивается в config.AWAKENING_GOLD_COST)
- Бонус: +50 к силе за каждое пробуждение
- После пробуждения XP пробуждения сбрасывается до 0

## Получение XP пробуждения

Игроки могут получить XP пробуждения через:
- 🥊 Дуэли: +100 XP за победу
- 📜 Квесты: +50-200 XP в зависимости от квеста
- 🔍 Поиск предметов: +20 XP

## Технические детали

### Callback handlers
- `upgrade_race` - показывает меню прокачки расы
- `confirm_upgrade_race` - подтверждает и выполняет прокачку
- `awaken` - показывает меню пробуждения расы
- `confirm_awaken` - подтверждает и выполняет пробуждение

### База данных

#### Прокачка расы
```sql
UPDATE players 
SET gold = gold - ?, 
    power = power + 20 
WHERE user_id = ?
```

#### Пробуждение расы
```sql
UPDATE players 
SET gold = gold - ?, 
    awakening_level = awakening_level + 1, 
    awakening_xp = 0, 
    power = power + 50 
WHERE user_id = ?
```

### Конфигурация (config.js)

```javascript
AWAKENING_XP_REQUIRED: 1000,  // XP для пробуждения
AWAKENING_GOLD_COST: 500,     // Стоимость пробуждения в золоте
```

## Формулы

### Стоимость прокачки расы
```
cost = 100 * (player.level + 1)
```

Примеры:
- Уровень 1: 200 золота
- Уровень 5: 600 золота
- Уровень 10: 1100 золота
- Уровень 20: 2100 золота

### Бонусы
- Прокачка расы: +20 силы (фиксированный бонус)
- Пробуждение расы: +50 силы (фиксированный бонус)

## Примеры использования

### Сценарий 1: Прокачка расы
1. Игрок открывает меню "⬆️ Улучшения"
2. Нажимает "⬆️ Прокачать расу"
3. Видит стоимость и текущее золото
4. Подтверждает прокачку
5. Получает +20 к силе, теряет золото
6. Видит сообщение об успехе

### Сценарий 2: Пробуждение расы
1. Игрок фармит XP пробуждения через дуэли и квесты
2. Накапливает 1000 XP пробуждения
3. Открывает меню "⬆️ Улучшения"
4. Нажимает "🌟 Пробудить расу"
5. Видит требования (1000 XP, 500 золота)
6. Подтверждает пробуждение
7. Получает +50 к силе, уровень пробуждения +1
8. XP пробуждения сбрасывается до 0

### Сценарий 3: Недостаточно ресурсов
1. Игрок пытается прокачать расу
2. Не хватает золота
3. Видит сообщение об ошибке с указанием нужной суммы
4. Возвращается в главное меню

## Баланс

### Прокачка расы
- Стоимость растет с уровнем игрока
- Бонус фиксированный (+20 силы)
- Нет ограничений на количество прокачек
- Подходит для постоянного улучшения

### Пробуждение расы
- Фиксированная стоимость (500 золота)
- Требует фарм XP пробуждения
- Ограничено 10 уровнями
- Дает больший бонус (+50 силы)
- Подходит для значительных улучшений

## Заключение

Система прокачки расы полностью реализована и интегрирована в игру. Игроки могут улучшать свою расу двумя способами:
1. Регулярная прокачка за золото (+20 силы)
2. Пробуждение за золото и XP пробуждения (+50 силы, до 10 уровней)

Система сбалансирована, имеет понятный интерфейс и полное логирование всех действий.
