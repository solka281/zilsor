## Преимущества

✅ **Читаемость** - легче воспринимать большие числа
✅ **Компактность** - меньше места в сообщениях
✅ **Профессионализм** - как в популярных играх
✅ **Универсальность** - работает для любых чисел
✅ **Гибкость** - настраиваемая точность

---

## Сравнение с другими методами

### .toLocaleString()
```javascript
1390000.toLocaleString('ru-RU')  // "1 390 000" - длинно
smartFormat(1390000)              // "1.39M" - компактно
```

### Обычный вывод
```javascript
`Золото: ${player.gold}`          // "Золото: 1390000" - нечитаемо
`Золото: ${smartFormat(player.gold)}` // "Золото: 1.39M" - отлично
```

---

## Настройка точности

```javascript
// По умолчанию 2 знака после запятой
formatNumber(1234567)     // "1.23M"

// Без десятичных
formatNumber(1234567, 0)  // "1M"

// 1 знак
formatNumber(1234567, 1)  // "1.2M"

// 3 знака
formatNumber(1234567, 3)  // "1.235M"
```

---

## Особые случаи

```javascript
// Отрицательные числа
smartFormat(-1500000)     // "-1.5M"

// Ноль
smartFormat(0)            // "0"

// Null/undefined
smartFormat(null)         // "0"
smartFormat(undefined)    // "0"

// Очень большие числа
smartFormat(1e15)         // "1Q"
```
