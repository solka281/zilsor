- `applyStatBonus()` - бонусы к характеристикам
- `applyItemBonus()` - бонусы к предметам

## Как тестировать

### Тест 1: Вампир - Кровопийство
1. Получить расу Вампир
2. Начать бой в лесу
3. Атаковать врага
4. Проверить сообщение: "🩸 Кровопийство! +X HP"
5. Убедиться что HP восстановилось

### Тест 2: Феникс - Возрождение
1. Получить расу Феникс
2. Начать бой с сильным врагом
3. Дождаться смерти (HP = 0)
4. Проверить сообщение: "🔥 ВОЗРОЖДЕНИЕ! Феникс восстал из пепла с X HP!"
5. Убедиться что бой продолжается

### Тест 3: Ангел - Божественная защита
1. Получить расу Ангел
2. Начать бой
3. Атаковать несколько раз
4. Дождаться сообщения: "✨ Божественная защита! +X HP"
5. Убедиться что HP восстановилось

### Тест 4: Демон - Адское пламя
1. Получить расу Демон
2. Начать бой
3. Атаковать до срабатывания способности
4. Проверить сообщение: "🔥 Адское пламя! Противник горит (X урона/ход)"
5. В следующих ходах проверить: "🔥 Горение: -X HP"

### Тест 5: Бог Войны - Божественная ярость
1. Получить расу Бог Войны
2. Начать бой
3. Атаковать до срабатывания способности
4. Проверить сообщение: "⚔️ БОЖЕСТВЕННАЯ ЯРОСТЬ! Неуязвимость на 2 хода!"
5. Убедиться что следующие 2 атаки врага не наносят урон

### Тест 6: Человек - Адаптация
1. Получить расу Человек
2. Запомнить текущий опыт
3. Победить врага в лесу
4. Проверить что опыт увеличился на 5% больше обычного

## Технические детали

### Структура battleContext

```javascript
// Для PvE боев
{
  phoenixRevived: false,
  godShield: 0,
  burnDamage: 0,
  burnTurns: 0
}

// Для PvP боев
{
  player1: {
    phoenixRevived: false,
    godShield: 0,
    burnDamage: 0,
    burnTurns: 0
  },
  player2: {
    phoenixRevived: false,
    godShield: 0,
    burnDamage: 0,
    burnTurns: 0
  }
}
```

### Порядок применения способностей

1. **Начало хода**: applyTurnEffects() - горение, яды
2. **Атака**: performAttack() → applyRaceAbility() - атакующие способности
3. **Защита**: applyDefensiveAbility() - защитные способности
4. **Смерть**: checkPhoenixRevive() - возрождение
5. **Конец боя**: applyExpBonus() - бонусы к опыту

## Баланс

Все способности сбалансированы по редкости:
- COMMON: 5-15% бонусы, 25-30% шанс
- RARE: 10-20% бонусы, 20-25% шанс
- EPIC: 15-80% бонусы, 15-20% шанс
- MYTHIC: Уникальные механики (возрождение, вампиризм)
- LEGENDARY: Экстремальные эффекты (x2 урон, неуязвимость), 8-10% шанс
- SECRET: Божественные эффекты (x2.5 урон + лечение), 15% шанс
