# Raid System Cooldown Fix

## Исправленные проблемы:

### ✅ 1. Проблема с интерфейсом "Доступные боссы"
**Проблема:** Когда все боссы были на кулдауне, показывалось окно "Доступные боссы" вместо обычного выбора босса с навигацией.

**Решение:**
- Полностью переработана функция `showRaidBossSelection()` в `bot.js`
- Теперь всегда показывается интерфейс выбора босса с навигацией ◀️ ▶️
- Для каждого босса индивидуально проверяется его кулдаун
- Кнопка меняется в зависимости от статуса босса:
  - `⚔️ Атаковать босса` - если босс готов
  - `⏰ Рейд пока недоступен` - если босс на кулдауне

### ✅ 2. Проблема с одинаковым кулдауном у всех боссов
**Проблема:** У всех боссов был одинаковый кулдаун, хотя у "Повелителя ветра" должно быть 2 часа, а у "Владыки тьмы" - 5 часов.

**Решение:**
- Добавлена новая функция `getNextRaidTimeForBoss(bossName, callback)` в `raids.js`
- Функция проверяет кулдаун конкретного босса, а не общий
- Обновлена функция `getNextRaidTime()` для работы с индивидуальными кулдаунами
- Автоматическое создание рейдов теперь выбирает только готовых боссов

## 🔧 Технические изменения:

### В `raids.js`:

#### Новая функция `getNextRaidTimeForBoss()`:
```javascript
function getNextRaidTimeForBoss(bossName, callback) {
  // Проверяем последний завершенный рейд с этим боссом
  db.get(`SELECT rh.*, rb.cooldown_hours 
          FROM raid_history rh
          LEFT JOIN raid_bosses rb ON rh.boss_name = rb.name
          WHERE rh.status = 'completed' AND rh.boss_name = ?
          ORDER BY rh.id DESC LIMIT 1`, [bossName], ...);
}
```

#### Обновленная функция `getNextRaidTime()`:
- Проверяет всех боссов индивидуально
- Возвращает список готовых боссов (`readyBosses`)
- Рассчитывает минимальное время до следующего рейда

#### Обновленная функция `checkAndCreateRaid()`:
- Использует список готовых боссов из `getNextRaidTime()`
- Выбирает случайного босса только из готовых
- Не создает рейды если все боссы на кулдауне

### В `bot.js`:

#### Полностью переработанная `showRaidBossSelection()`:
- Убрана проверка общего времени рейда
- Добавлена индивидуальная проверка каждого босса
- Всегда показывается навигация между боссами
- Адаптивные кнопки в зависимости от статуса босса

## 📋 Логика работы:

### Индивидуальные кулдауны:
1. **Повелитель ветра**: 2 часа после завершения рейда с ним
2. **Владыка тьмы**: 5 часов после завершения рейда с ним
3. Каждый босс имеет независимый кулдаун

### Интерфейс выбора босса:
1. Всегда показывается навигация ◀️ ▶️ между боссами
2. Для каждого босса проверяется его индивидуальный кулдаун
3. Кнопки адаптируются под статус конкретного босса
4. Показывается точное время до готовности каждого босса

### Автоматическое создание рейдов:
1. Система проверяет всех боссов каждые 5 минут
2. Создает рейд только с готовыми боссами
3. Выбирает случайного босса из доступных
4. Не создает рейды если все боссы на кулдауне

## 🎯 Результат:

### ✅ Исправлено:
- Интерфейс всегда показывает выбор босса с навигацией
- Каждый босс имеет свой индивидуальный кулдаун
- Повелитель ветра: 2 часа кулдауна
- Владыка тьмы: 5 часов кулдауна
- Автоматические рейды создаются только с готовыми боссами

### 🎮 Пользовательский опыт:
- Игроки всегда могут листать между боссами
- Видят точное время до готовности каждого босса
- Понимают какой босс когда будет доступен
- Могут планировать участие в рейдах

### 🔄 Системная логика:
- Каждый босс работает независимо
- Кулдауны не пересекаются между боссами
- Автоматическое создание стало более умным
- Нет ложных уведомлений о готовности рейдов

Теперь система рейдов работает корректно с индивидуальными кулдаунами и правильным интерфейсом!
